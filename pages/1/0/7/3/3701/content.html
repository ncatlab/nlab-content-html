Summary

This page gives an elementary description of the locale of real numbers, that is the localic real line. The development is manifestly constructive and even predicative over the natural numbers (although we are somewhat careless with the language and do not always point out when a set may predicatively be a proper class). Ideally, we will show that our construction satisfies the seven ‚Äòheadline properties‚Äô of the real line described by Bauer & Taylor (although so far we cover only the Heine‚ÄìBorel theorem).

The exposition here is pretty much my own work, although of course the basic ideas are well known to many. In particular, the technical zigzag lemma is mine (but it is not a very deep result!).

Idea

To describe the locale of the real numbers, we need first of all to describe what an open in the real line is. The key insight is that an open set is determined by the open intervals of rational numbers that it contains. This is analogous to the key insight of Richard Dedekind's definition of real number: that a point in the real line is determined by which open intervals of rational numbers that it belongs to. (Once you're talking about rational numbers, things are manageable.)

The zigzag lemma

We will often have occasion to consider a zigzag as in (1), but where the various zigs are taken relative to different opens. The frame of opens

The opens form a sub-poset of the power set ùí´(‚Ñö√ó‚Ñö)\mathcal{P}(\mathbb{Q} \times \mathbb{Q}). This poset is in fact a frame, as we will now show. (It is not a subframe of the power set, since the joins are different. It is a sub-inflattice of the power set, although this seems to be a red herring at least for infinitary meets, since those are not part of the frame structure that we need.)

The top open, denoted ‚Ñù\mathbb{R}, is the binary relation which is always true. Given two opens GG and HH, their meet in the poset of opens, denoted G‚à©HG \cap H, is simply their conjunction, that is their intersection as subsets of ‚Ñö√ó‚Ñö\mathbb{Q} \times \mathbb{Q}. Intuitively, this all works because an open interval will be contained in the intersection of a family of open sets if and only if it is contained in each individual open set.

The bottom open, denoted ‚àÖ\empty, is the binary relation ‚â•\geq. That is, (a,b)‚äÜ‚àÖ(a,b) \subseteq \empty iff a‚â•ba \geq b. It is easy to check that this is an open; it precedes every open by property (1). Intuitively, this corresponds to the empty subset of the real line; (a,b)(a,b) is empty if and only if a‚â•ba \geq b. However, note that ‚àÖ\empty is not the empty subset of ‚Ñö√ó‚Ñö\mathbb{Q} \times \mathbb{Q}; the notation follows our topological intuition rather than the algebra of relations.

Open intervals as opens

Given rational numbers aa and bb, the open interval (a,b)(a,b) may itself be interpreted as an open in the real line, also denoted (a,b)(a,b), as follows: let (c,d)‚äÜ(a,b)(c,d) \subseteq (a,b) hold if every rational number strictly between cc and dd (in that order) is also strictly between aa and bb (in that order). There is now a third way to interpret ‚Äò(c,d)‚äÜ(a,b)(c,d) \subseteq (a,b)‚Äô; interpreting both intervals as opens in the real line, this states that the first is contained in the second. Notice that (a,b)=‚àÖ(a,b) = \empty whenever a‚â•ba \geq b.

We can actually generalise this somewhat. Open sets, closed sets, and points

We think of each open as defining an open subset of the real line, but we can equally well think of it as defining a closed subset. The difference between these perspectives is reflected in complementary criteria for when a real number belongs to the set. So to make sense of this, we must identify the points of the real line.

Recall that a real number may be defined as a pair (L,U)(L,U) of inhabited subsets of ‚Ñö\mathbb{Q} satisfying the following properties:

  1. If a‚ààLa \in L, then a<ba \lt b for some b‚ààLb \in L.
  2. If b‚ààUb \in U, then a<ba \lt b for some a‚ààUa \in U.
  3. If a‚ààLa \in L and b‚ààUb \in U, then a<ba \lt b.
  4. If a<ba \lt b, then a‚ààLa \in L or b‚ààUb \in U.

We define a point of the real line to be a real number in this sense. Given such a point x=(L,U)x = (L,U) and a rational number aa, we write a<xa \lt x to mean that a‚ààLa \in L and a>xa \gt x to mean that a‚ààUa \in U. If we wish to refer to LL and UU directly, we may call LL the lower set of xx and UU the upper set.

Given a point xx and an open GG, we say that xx belongs to GG, written x‚ààGx \in G, if (a,b)‚äÜG(a,b) \subseteq G for some a<xa \lt x and b>xb \gt x; that is, GG contains an interval from some element of the lower set to some element of the upper set of xx. We have x‚àà‚Ñùx \in \mathbb{R} since its lower and upper sets are inhabited. If x‚ààGx \in G and x‚ààHx \in H, with (a,b)‚äÜG(a,b) \subseteq G and (c,d)‚äÜH(c,d) \subseteq H, then (max(a,c),min(b,d))‚äÜG‚à©H\big(\max(a,c), \min(b,d)\big) \subseteq G \cap H, so x‚ààG‚à©Hx \in G \cap H; note that this argument fails for infinitary intersections. (The converse, that x‚ààGx \in G and x‚ààHx \in H if x‚ààG‚à©Hx \in G \cap H, is immediate.) Dually, suppose that x‚àà‚ãÉ kG kx \in \bigcup_k G_k, as shown by some zigzag (since a‚â•ba \geq b is impossible when a<xa \lt x and b>xb \gt x, by 3). Applying condition (4) of the definition of real number to each zag, we have a i +<xa_{i^+} \lt x or b i>xb_i \gt x, for each i<ni \lt n. Checking all 2 n‚àí12^{n-1} possibilities, and knowing that a 1<xa_1 \lt x and b n>xb_n \gt x in any case, we must have a i<xa_i \lt x and b i>xb_i \gt x for some ii. Therefore, each point defines a completely prime filter on the frame of all opens, which is the definition of a point in general locale theory. Given a point xx and an open GG, we say that xx co-belongs to GG, written x‚àâGx \notin G, if we never have a<xa \lt x, b>xb \gt x, and (a,b)‚äÜG(a,b) \subseteq G, which is precisely the negation of the property that x‚ààGx \in G. A related question is whether we can reconstruct GG from the set of points which belong to it. This should be equivalent to the fan theorem, which is classically true and also accepted by Brouwer's school of intuitionism, but refuted in the Russian school in which all real numbers are assumed to be computable. (I should check this.) Arguably, the real lesson of these logical technicalities is that we should remember that opens, not points, are the fundamental concept in a locale.

The Heine‚ÄìBorel theorem

The classical Heine‚ÄìBorel theorem, as a statement about sets of real numbers, may fail constructively; this is related to the comments above about the fan theorem. But the beauty of the localic approach is that Heine‚ÄìBorel necessarily holds when interpreted as a statement about opens in the locale of real numbers. To state the theorem, we must define what it means for a collection of opens to cover the unit interval. We will give an ad-hoc definition, but this may also be derived from the general theory of closed sublocales which allows us to interpret the unit interval as a compact locale in its own right.

Definition

An open cover of the unit interval is a collection ùíû\mathcal{C} of opens in the real line such that ‚Ñù\mathbb{R} is the join of (‚àí‚àû,0)(-\infty,0), (1,‚àû)(1,\infty), and the elements of ùíû\mathcal{C}.

(Heine‚ÄìBorel)

Every open cover of the unit interval has a finite subcover.

Proof

If this equals ‚Ñù\mathbb{R}, then in particular (‚àí2,3)‚äÜJ(-2,3) \subseteq J, and since (‚àí1,2)‚ãê(‚àí2,3)(-1,2) \Subset (-2,3), we get a corresponding zigzag Œ∂\zeta involving finitely many zigs using finitely many of the elements of ùíû\mathcal{C}. ¬†As a classifying locale

The locale of real numbers is the classifying locale of the geometric theory of Dedekind real numbers.

¬†Defining functions in the locale of real numbers

There are two general ways to define functions on the locale of real numbers, via the rational numbers and via the Dedekind real numbers.

Using the rational numbers

The first approach uses existing functions already defined on the rational numbers ‚Ñö\mathbb{Q}. Suppose that one has a locally uniformly continuous function f:‚Ñö‚Üí‚Ñöf:\mathbb{Q} \to \mathbb{Q}. Then one could extend ff to a continuous map f‚Ä≤:‚Ñù‚Üí‚Ñùf':\mathbb{R} \to \mathbb{R} on the locale of real numbers.

Using the Dedekind real numbers.

The other approach is via using existing functions defined on the Dedekind real numbers ‚Ñù\mathbb{R}. In particular, given any well-defined continuous function x‚Ü¶f(x)x \mapsto f(x) on the Dedekind real numbers, and a geometric morphism F:Set‚ÜíSh(‚Ñù D)F:\mathrm{Set} \to \mathrm{Sh}(\mathbb{R}_D) from Set to the sheaf topos Sh(‚Ñù D)\mathrm{Sh}(\mathbb{R}_D), if for all Dedekind real numbers x‚àà‚Ñù Dx \in \mathbb{R}_D, f(F *(x))=F *(f(x))f(F^*(x)) = F^*(f(x)), then the function x‚Ü¶f(x)x \mapsto f(x) lifts to a function x‚Ü¶fÀú(x)x \mapsto \widetilde{f}(x) on the locale of real numbers.

This allows us to first define real-valued functions normally on the set of Dedekind real numbers, and then lift them up to a real-valued function on the locale of real numbers, thus allowing us to define functions such as the exponential function, the sine, and the cosine which can‚Äôt be directly defined using functions on the rational numbers.

Arithmetic

Significantly, since the field operations on the discrete locale of rational numbers are all locally uniformly continuous on the ‚Ñö\mathbb{Q}, and the field operations are well-defined and continuous on the Dedekind real numbers ‚Ñù D\mathbb{R}_D, the field operations in both cases lift up to a field structure on the locale of real numbers, allowing one to do basic arithmetic on ‚Ñù\mathbb{R}.
