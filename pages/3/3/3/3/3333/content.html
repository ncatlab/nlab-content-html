Idea

In an (∞,1)-category CC with (∞,1)(\infty,1)-pullbacks, the free loop space object ℒX\mathcal{L}X of any object XX – also called the inertia groupoid – is an object that behaves as if its generalized elements are loops in XX, morphisms between generalized elements homotopies of loops, and so on.

For the case that C=C = Top this reproduces the ordinary notion of free loop spaces of topological spaces.

Over each fixed element x∈Xx \in X, the free loop space object ℒX\mathcal{L}X looks like the based loop space object Ω xX\Omega_x X of XX.

Free loop space objects come naturally equipped with various structures of interest, such as a categorical circle action. The cohomology of ℒX\mathcal{L}X is Hochschild cohomology or cyclic cohomology of function algebras C(X)C(X) on XX. The categorical circle action induces differentials on these cohomologies, identifying them, in suitable cases, with algebras of Kähler differential forms on XX.

Definition

In higher category theory

There are various equivalent ways to define the free loop space object.

Let CC be an (∞,1)-category. Recall that every (∞,1)(\infty,1)-category is enriched over ∞Grpd, in that there is a hom-space Map(X,Y)∈∞GpdMap(X,Y) \in \infty Gpd for any X,Y∈CX,Y\in C. This enables us to define the power of an object X∈CX\in C by any ∞\infty-groupoid KK, as an object X K∈CX^K \in C together with a map K→Map(X K,X)K \to Map(X^K,X) inducing equivalences Map(Y,X) K≃Map(Y,X K)Map(Y,X)^K \simeq Map(Y,X^K) for all Y∈CY\in C (where Map(Y,X) KMap(Y,X)^K denotes the mapping-space from KK to Map(Y,X)Map(Y,X) in ∞Gpd\infty Gpd), if such exists.

Definition

The free loop space object of X∈CX\in C is the power ℒX=X S 1\mathcal{L}X = X^{S^1}, if it exists, where S 1S^1 denotes the homotopical circle.

This can also be written in terms of “conical” limits in CC. Most commonly, if we note that S 1S^1 is the pushout of two copies of *\ast under *⊔*\ast\sqcup \ast, and that Map(*,X)≃XMap(\ast,X) \simeq X while Map(*⊔*,X)≃X×XMap(\ast\sqcup \ast ,X) \simeq X\times X, we find that X S 1X^{S^1} is the pullback of XX and XX over X×XX\times X:

Definition

In an (∞,1)-category CC with (∞,1)-pullbacks, for X∈CX \in C an object, its free loop space object ℒX\mathcal{L}X is the (∞,1)(\infty,1)-pullback of the diagonal along itself

ℒX → X ↓ ↓ (Id,Id) X →(Id,Id) X×X. \array{ \mathcal{L} X &\to& X \\ \downarrow && \downarrow^{\mathrlap{(Id,Id)}} \\ X &\stackrel{(Id,Id)}{\to}& X \times X } \,.
Remark

This is the (∞,1)(\infty,1)-categorical span trace of the identity-span

ℒX=Tr( X Id↙ ↘ Id X X). \mathcal{L}X = Tr \left( \array{ && X \\ & {}^{\mathllap{Id}}\swarrow && \searrow^{\mathrlap{Id}} \\ X &&&& X } \right) \,.

We can also use the fact that S 1S^1 is the homotopy coequalizer of *⇉*\ast \rightrightarrows \ast:

Definition

The free loop space object of X∈CX\in C is the homotopy equalizer of two copies of the identity map X⇉XX \rightrightarrows X.

In homotopy type theory

In the literature (see below) when the free loop space object ℒX\mathcal{L}X is defined as the homotopy pullback X× X×XXX\times_{X\times X} X, it is sometimes described heuristically as: “a point of ℒX\mathcal{L}X is a choice of making two points of XX equal in two ways.” In terms of homotopy type theory this heuristics becomes a theorem. In that higher categorical logic we have the expression

ℒX ≔{x,y:X|(x=y)and(x=y)} ={x,y:X|(x,x)=(y,y)}. \begin{aligned} \mathcal{L}X & \coloneqq \left\{ x,y : X \;|\; (x = y) \, and\, (x = y) \right\} \\ & = \left\{ x,y : X \;|\; (x,x) = (y,y) \right\} \end{aligned} \,.

Here on the right we have

  1. the dependent sum;

  2. over the identity type Id(X×X)Id (X \times X);

  3. of the product type X×XX \times X.

See the discussion at homotopy pullback in the section Construction in homotopy type theory for how this is equivalent to the previous definition.

Now since ∑ y:X(x=y)\sum_{y:X} (x=y) is contractible, the above type is equivalent to

∑ x:X(x=x) \sum_{x:X} (x=x)

i.e. the type of points that are equal to themselves (in a specified, not necessarily reflexivity, way). This corresponds to the other definitions, as a homotopy equalizer or a powering by S 1S^1.

Properties

Models by homotopy pullbacks

To see what the definition of a free loop space object amounts to in more detail, assume that the (∞,1)-category is modeled by a homotopical category, say for simplicity a category of fibrant objects, for instance the full subcategory on fibrant objects of a model category.

Then following the discussion at homotopy pullback and generalized universal bundle we can compute the about (∞,1)(\infty,1)-pullback X× X×X hXX\times_{X\times X}^h X as the ordinary limit

ℒX → → X ↓ ↓ (Id,Id) (X×X) I →(d 0,d 0) X×X ↓ (d 1,d 1)↓ X →(Id,Id) X×X, \array{ \mathcal{L}X &\to& &\to& X \\ \downarrow && && \downarrow^{\mathrlap{(Id,Id)}} \\ && (X \times X)^I &\stackrel{(d_0,d_0)}{\to}& X \times X \\ \downarrow && {}^{\mathllap{(d_1,d_1)}}\downarrow \\ X &\stackrel{(Id,Id)}{\to} & X \times X \,, }

where (X×X) I(X\times X)^I is a path space object for X×XX \times X. At least if we have the structure of a model category we may take (X×X) I=X I×X I(X \times X)^I = X^I \times X^I for a path space object X IX^I of XX.

From this description one sees that ℒX\mathcal{L}X is built from pairs of paths in XX with coinciding endpoints, that are glued at their coinciding endpoint . So the loops here are all built from two semi-ciricle paths.

Relation to based loop space object

The fiber of ℒX\mathcal{L}X over a point x:*→Xx : {*} \to X is the corresponding (based) loop space object Ω xX\Omega_x X of XX: we have an (∞,1)(\infty,1)-pullback diagram

Ω xX → ℒX → X ↓ ↓ ↓ (Id,Id) * →x X →(Id,Id) X×X. \array{ \Omega_x X &\to& \mathcal{L}X &\to& X \\ \downarrow && \downarrow && \downarrow^{\mathrlap{(Id,Id)}} \\ {*} &\stackrel{x}{\to} & X &\stackrel{(Id,Id)}{\to}& X\times X } \,.

To see this, use that homotopy pullbacks paste to homotopy pullbacks, so that the outer pullback is modeled by the ordinary limit

Ω x I∨IX → → X ↓ ↓ (Id,Id) (X×X) I →(d 0,d 0) X×X ↓ (d 1,d 1)↓ * →(x,x)) X×X, \array{ \Omega_x^{I \vee I}X &\to& &\to& X \\ \downarrow && && \downarrow^{\mathrlap{(Id,Id)}} \\ && (X \times X)^I &\stackrel{(d_0,d_0)}{\to}& X \times X \\ \downarrow && {}^{\mathllap{(d_1,d_1)}}\downarrow \\ {*} &\stackrel{(x,x))}{\to} & X \times X \,, }

which builds based loops on XX from two consecutive paths, the first starting at the basepoint xx, the second ending there. This is weakly equivalent Ω xX=Ω x IX→≃Ω I∨IX\Omega_x X = \Omega^I_x X \stackrel{\simeq}{\to} \Omega^{I \vee I} X to the based loop space object Ω xX\Omega_x X built from just the path space object X IX^I with a single copy of II, by standard arguments as for instance form page 12 on in

  • Kenneth Brown, Abstract Homotopy Theory and Generalized Sheaf Cohomology .

Action on objects of the slice

The free loop space object ℒX\mathcal{L} X is a group object in the slice (∞,1)-category C/XC/X, and has a canonical action on all objects of C/XC/X.

Intuitively, the group structure comes from composition and inversion of loops. When the free loop space is expressed as a power X S 1X^{S^1}, this group structure comes from the canonical cogroup structure on S 1S^1. In homotopy type theory, it is literally concatenation of paths. And when ℒX\mathcal{L}X is expressed as the pullback X× X×XXX\times_{X\times X} X, the group multiplication can be obtained by considering the following pasting square:

ℒX× XℒX → ℒX → X ↓ ↓ ↓ ℒX → X → X×X ↓ ↓ ↓ id X → X×X →id X×X\array{ \mathcal{L}X \times_X \mathcal{L}X & \to & \mathcal{L} X & \to & X\\ \downarrow && \downarrow && \downarrow \\ \mathcal{L} X & \to & X & \to & X\times X \\ \downarrow && \downarrow && \downarrow^{id} \\ X & \to & X\times X & \xrightarrow{id} & X\times X }

Here the bottom-left and top-right squares are the pullback defining ℒX\mathcal{L}X, the top-left square is the pullback defining ℒX× XℒX\mathcal{L}X \times_X\mathcal{L}X, and the bottom-right square commutes but is not a pullback. By the universal property of the pullback square defining ℒX\mathcal{L}X, this square factors through it uniquely, giving the composition map ℒX× XℒX→ℒX\mathcal{L}X \times_X \mathcal{L}X\to \mathcal{L}X. Similarly but more simply, the inversion map ℒX→ℒX\mathcal{L}X\to \mathcal{L}X comes from transposing its defining pullback square and factoring it through itself.

Now consider Y→XY\to X an object of C/XC/X. There is a canonical projection ℒX× XY→Y\mathcal{L}X \times_X Y \to Y, which is not the action of ℒX\mathcal{L}X on YY, but it almost is. In fact since this projection is a morphism in the “homotopy” slice C/XC/X, it consists not just of a morphism in CC but a homotopy witnessing that a certain triangle commutes, which is equivalently the homotopy in the left-hand commutative square below (which is the pullback defining ℒX× XY\mathcal{L}X \times_X Y):

ℒX× XY → ℒX →id ℒX ↓ ↓ ↓ Y → X →id X\array{ \mathcal{L}X \times_X Y & \to & \mathcal{L} X & \xrightarrow{id} & \mathcal{L}X \\ \downarrow && \downarrow && \downarrow \\ Y & \to & X & \xrightarrow{id} & X}

Pasting this with the right-hand commutative square, which is the canonical automorphism of the projection ℒX→X\mathcal{L}X\to X, we obtain a different homotopy witnessing a different morphism ℒX× XY→Y\mathcal{L}X \times_X Y \to Y in C/XC/X (with the same underlying morphism in CC), and this is the action of ℒX\mathcal{L}X on YY.

The definitiong of the right-hand commutative square above may not be obvious. It is clear when we write ℒX=X S 1\mathcal{L}X = X^{S^1}; when we write ℒX=X× X×XX\mathcal{L}X = X\times_{X\times X} X it can be obtained as the following pasting square, where pp and qq are the two projections in the defining pullback of ℒX\mathcal{L}X:

ℒX →p X → X×X →π 1 X id↓ ↓ id ↓ ℒX →q X → X×X →π 1 X id↓ ↓ id ↓ ℒX →q X → X×X →π 2 X id↓ ↓ id ↓ ℒX →p X → X×X →π 2 X id↓ ↓ id ↓ ℒX →p X → X×X →π 1 X \array{ \mathcal{L}X & \xrightarrow{p} & X & \to & X\times X & \xrightarrow{\pi_1} & X\\ ^{id}\downarrow && && \downarrow^{id} && \downarrow \\ \mathcal{L}X & \xrightarrow{q} & X & \to & X\times X & \xrightarrow{\pi_1} & X\\ ^{id}\downarrow && \downarrow^{id} && && \downarrow \\ \mathcal{L}X & \xrightarrow{q} & X & \to & X\times X & \xrightarrow{\pi_2} & X\\ ^{id}\downarrow && && \downarrow^{id} && \downarrow \\ \mathcal{L}X & \xrightarrow{p} & X & \to & X\times X & \xrightarrow{\pi_2} & X\\ ^{id}\downarrow && \downarrow^{id} && && \downarrow \\ \mathcal{L}X & \xrightarrow{p} & X & \to & X\times X & \xrightarrow{\pi_1} & X }

To extend these structures to a coherent ∞\infty-group structure and a coherent ∞\infty-action, see for instance this MO answer.

In an (∞,1)(\infty,1)-topos

We consider now the case that C=HC = \mathbf{H} is an (∞,1)-topos (of (∞,1)-sheaves/∞-stacks). This comes canonically with its terminal global sections (∞,1)-geometric morphism

(LConst⊣Γ):H→Γ←LConst∞Grpd. (LConst \dashv \Gamma) : \mathbf{H} \stackrel{\overset{LConst}{\leftarrow}}{\underset{\Gamma}{\to}} \infty Grpd \,.

In this case we can reformulate the power definition ℒX=X S 1\mathcal{L}X = X^{S^1} using a version of S 1S^1 that is an object of H\mathbf{H} itself.

As a mapping space object

Definition/Proposition

Write

S 1∈Top≃∞Grpd↪LConstH S^1 \in Top \simeq \infty Grpd \stackrel{LConst}{\hookrightarrow} \mathbf{H}

for the circle. In Top this is the usual topological circle. In ∞Grpd this is (the homotopy type of) the fundamental ∞-groupoid of the topological circle. We may think of this as the (∞,1)-pushout

S 1≃*∐ *∐** S^1 \simeq * \coprod_{* \coprod *} *

hence as the universal cocone

*∐* → * ↓ ⇙ ≃ ↓ * → S 1 \array{ * \coprod * &\to& * \\ \downarrow &\swArrow_{\simeq}& \downarrow \\ * &\to& S^1 }

in ∞Grpd\infty Grpd.

In H\mathbf{H} we still write S 1S^1 for the constant ∞-stack on this, the image of this under LConstLConst. Since LConstLConst is a left adjoint and hence preserves this poushout, there is no risk of confusion.

Proof

To see that the given (∞,1)(\infty,1)-pushout indeed produces the circle, we use the standard model structure on simplicial sets sSet QuillensSet_{Quillen} to present ∞Grpd. In sSet QuillensSet_{Quillen} the (∞,1)(\infty,1)-pushout is computed by the homotopy pushout. By general facts about this, it may be computed as an ordinary pushout in sSet once we pass to an equivalent pushout diagram in which at least one morphism is a monomorphism. This is the case for

*∐* → * ↓ ↓ Δ[1] → Δ[1]/∂Δ[1]. \array{ * \coprod * &\to& * \\ \downarrow && \downarrow \\ \Delta[1] &\to& \Delta[1]/\partial \Delta[1] } \,.
Remark

Informally the (∞,1)(\infty,1)-pushout *∐ *∐*** \coprod_{* \coprod *} * may be thought of as

  • the disjoint union of two points * 1*_1, * 2*_2;

  • equipped with two non-equivalent abstract homotopies between them

    S 1={ ↗↘ ≃ * 1 * 2 ↘↗ ≃}. S^1 = \left\{ \array{ & \nearrow \searrow^{\mathrlap{\simeq}} \\ *_1 && *_2 \\ & \searrow \nearrow_{\mathllap{\simeq}} } \right\} \,.

This equivalent way of modelling the circle not as a single point with an automorphism, but as two points with two isomorphisms is what connects directly to the definition of the free loop space object. This we now come to. It is also the fundamenal source of the basic structure of Hochschild (co)homology (as discussed there).

Lemma

Every (∞,1)-topos is a cartesian closed (∞,1)-category: we have for every object X∈HX \in \mathbf{H} an internal hom-(∞,1)-functor

[X,−]:H→H. [X,-] : \mathbf{H} \to \mathbf{H} \,.

Proof

This is discussed at (∞,1)-topos in the section Closed monoidal structure.

Proposition

There is a natural equivalence

ℒX≃[S 1,X]. \mathcal{L}X \simeq [S^1 , X] \,.
Proof

This follows from the above by the fact (see closed monoidal structure on (∞,1)-toposes) that the internal hom in an (∞,1)(\infty,1)-topos preserves finite colimits in its first argument and satisfies

[*,X]≃X. [*,X] \simeq X \,.

This yields

[S 1,X] ≃[*∐ *∐**,X] ≃[*,X]× [*∐*,X][*,X] ≃X× X×XX ≃ℒX. \begin{aligned} [S^1 ,X] &\simeq [* \coprod_{* \coprod *} *, X] \\ & \simeq [*,X] \times_{[* \coprod *, X]} [*,X] \\ & \simeq X \times_{X \times X} X \\ & \simeq \mathcal{L}X \,. \end{aligned}
Corollary

We have that the free loop space object of X∈HX \in \mathbf{H} is equivalently the powering of XX by the ∞\infty-groupoid S 1S^1:

ℒX≃X S 1 \mathcal{L} X \simeq X^{S^1}
Proof

Follows by the above from the equivalence [LConstS 1,X]≃X S 1[LConst S^1 , X] \simeq X^{S^1} discussed at (∞,1)-topos.

Intrinsic circle action

By precomposition, the automorphism 2-group of the circle S 1S^1 acts on free loop space of an object X∈HX \in \mathbf{H}

Aut(S 1)×[S 1,X]→[S 1,S 1]×[S 1,X]→∘[S 1,X]. Aut(S^1) \times [S^1, X] \to [S^1, S^1] \times [S^1, X] \stackrel{\circ}{\to} [S^1, X] \,.
Proposition

The connected component of [S 1,S 1][S^1,S^1] on the identity is equivalent to S 1S^1

[S 1,S 1] Id≃S 1 [S^1 , S^1]_{Id} \simeq S^1
Definition

We say that

S 1×[S 1,X]≃[S 1,S 1] Id×[S 1,X]→∘[S 1,X] S^1 \times [S^1, X] \simeq [S^1, S^1]_{Id} \times [S^1, X] \stackrel{\circ}{\to} [S^1, X]

is the intrinsic circle action on the free loop space object.

Proof

We spell out in detail what this action looks like. The reader should thoughout keep the homotopy hypothesis-equivalence, (|−|⊣Π):Top≃∞Grpd(|-| \dashv \Pi) : Top \simeq \infty Grpd in mind.

We may realize the circle S 1∈S^1 \in Top under Π:Top≃∞Grpd\Pi : Top \simeq \infty Grpd as the [delooping]] groupoid Bℤ\mathbf{B}\mathbb{Z} of the additive group ℤ\mathbb{Z} of integers

The automorphism 2-group of this object is the functor groupoid

Aut Grpd(Bℤ) Aut_{Grpd}(\mathbf{B}\mathbb{Z})

whose objects are invertible functors Bℤ→Bℤ\mathbf{B}\mathbb{Z} \to \mathbf{B}\mathbb{Z} and whose morphisms are natural transformations between these.

The functors Bℤ→Bℤ\mathbf{B}\mathbb{Z} \to \mathbf{B}\mathbb{Z} correspond bijectively to group homomorphisms ℤ→ℤ\mathbb{Z} \to \mathbb{Z}, hence to multiplication by n∈ℤn\in\mathbb{Z}

[n]:Bℤ→Bℤ [n] : \mathbf{B}\mathbb{Z} \to \mathbf{B}\mathbb{Z}
(•→k•)↦(•→n⋅k•). (\bullet \stackrel{k}{\to} \bullet) \mapsto (\bullet \stackrel{n\cdot k}{\to} \bullet).

Natural transformations between two such endomorphisms are given by a component ℓ∈ℤ\ell \in \mathbb{Z} such that all diagrams

• →ℓ • n⋅k↓ ↓ n′⋅k • →ℓ • \array{ \bullet &\stackrel{\ell}{\to}& \bullet \\ {}^{\mathllap{n\cdot k}}\downarrow && \downarrow^{\mathrlap{n' \cdot k}} \\ \bullet &\stackrel{\ell}{\to}& \bullet }

commute in Bℤ\mathbf{B}\mathbb{Z}. This can happen only for n=n′n = n', but then it happens for arbitrary ℓ\ell.

In other words we have

Aut(Bℤ)≃∐ [n]∈ℤ ×Bℤ. Aut(\mathbf{B}\mathbb{Z}) \simeq \coprod_{[n] \in \mathbb{Z}^\times}\mathbf{B}\mathbb{Z} \,.

and

Aut Id(Bℤ)≃Bℤ. Aut_{Id}(\mathbf{B}\mathbb{Z}) \simeq \mathbf{B}\mathbb{Z} \,.

The object [n][n] corresponds to the self-mapping of the circle that fixes the basepoint and has winding number n∈ℤn\in\mathbb{Z}. The transformation ℓ\ell corresponds then to a rigid rotation of the loop by ℓ\ell full circles

Notably for n=1n = 1 and k=1k = 1 we may think of the diagram

• →ℓ • 1↓ ↓ 1 • →ℓ • \array{ \bullet &\stackrel{\ell}{\to}& \bullet \\ {}^{\mathllap{1}}\downarrow && \downarrow^{\mathrlap{1}} \\ \bullet &\stackrel{\ell}{\to}& \bullet }

as depicting the unit loop around the circle (on the left, say) and the result of translating its basepoint ℓ\ell-times around the circle (the rest of the diagram). Of course since we are using a model of S 1S^1 with a single object here, every rotation of the loop is a full circle rotation, which is a bit hard to see.

Exercise: spell out the above discussion analogously for the equivalent model given by the fundamental groupoid Π 1(S 1)\Pi_1(S^1) of the standard circle. The is the groupoid with S Top 1S^1_{Top} as its set of objects homotopy classes of paths in the circle as morphisms. In this model things look more like one might expect from a circle action. Notice that Bℤ\mathbf{B}\mathbb{Z} is the skeleton of Π 1(S 1)\Pi_1(S^1).

Example

Consider H=\mathbf{H} = ∞Grpd, GG a group and X=BGX = \mathbf{B}G the delooping groupoid. Then ℒX=G// AdG\mathcal{L}X = G//_{Ad}G (as discussed in detail below). A morphism (g→hAd ha)(g \stackrel{h}{\to} Ad_h a) in G//GG//G corresponds to a natural transformation

↗↘ g Bℤ ⇓ h BG ↘↗ Ad hg. \array{ & \nearrow \searrow^{\mathrlap{g}} \\ \mathbf{B}\mathbb{Z} &\Downarrow^{h}& \mathbf{B}G \\ & \searrow \nearrow_{\mathrlap{Ad_h g}} } \,.

Precomposing this with the automorphism ℓ\ell of the object [n][n] in END(Bℤ)END(\mathbf{B}\mathbb{Z})

↗↘ n Bℤ ⇓ ℓ Bℤ ↘↗ n \array{ & \nearrow \searrow^{\mathrlap{n}} \\ \mathbf{B}\mathbb{Z} &\Downarrow^{\ell}& \mathbf{B}\mathbb{Z} \\ & \searrow \nearrow_{\mathrlap{n}} }

produces the new transformation

↗↘ n ↗↘ g Bℤ ⇓ ℓ Bℤ ⇓ h BG ↘↗ n ↘↗ Ad hg. \array{ & \nearrow \searrow^{\mathrlap{n}} && \nearrow \searrow^{\mathrlap{g}} \\ \mathbf{B}\mathbb{Z} &\Downarrow^{\ell}& \mathbf{B}\mathbb{Z} &\Downarrow^{h}& \mathbf{B}G \\ & \searrow \nearrow_{\mathrlap{n}} && \searrow \nearrow_{\mathrlap{Ad_h g}} } \,.

By the rules of horizontal composition of natural transformations, this is the transformation whose component naturality square on (•→1•)(\bullet \stackrel{1}{\to} \bullet) in Bℤ\mathbf{B}\mathbb{Z} is the diagram

• →g ℓ • →h • g n↓ g n↓ ↓ Ad hg n • →g ℓ • →h • \array{ \bullet &\stackrel{g^\ell}{\to}& \bullet &\stackrel{h}{\to}&\bullet \\ {}^{\mathllap{g^{n}}}\downarrow && {}^{g^n}\downarrow && \downarrow^{\mathrlap{Ad_h g^n}} \\ \bullet &\underset{g^\ell}{\to}& \bullet &\underset{h}{\to}&\bullet }

in Bℤ\mathbf{B}\mathbb{Z}, hence the morphism (g n→g ℓhAd hg n)(g^n \stackrel{g^{\ell} h}{\to} Ad_h g^n) in G// AdGG//_{Ad}G. In particular, the categorical circle action is

ℓ:(g→hAd hg)↦(g→g ℓhAd hg). \ell:(g \stackrel{h}{\to} Ad_h g)\mapsto (g \stackrel{g^{\ell} h}{\to} Ad_h g).

Examples

Free topological loop spaces

In Top the notion of free loop space objects reproduces the standard notion of topological free loop spaces.

Details for ℒBG\mathcal{L} \mathbf{B}G

Let the ambient (∞,1)-category be ∞Grpd, let GG be an ordinary group and BG\mathbf{B}G its one-object delooping groupoid.

Proposition

We have that the loop groupoid

ℒBG≃G// AdG, \mathcal{L} \mathbf{B}G \simeq G//_{Ad} G \,,

the action groupoid of the adjoint action of GG on itself.

Proof

We spell this out in full pedestrian detail, as a little exercise in computing homotopy pullbacks.

We have that the path space object is BG I=[I,BG]\mathbf{B}G^I = [I,\mathbf{B}G] – the functor groupoid, where II is the free groupoid I={a→≃b}I = \{a \stackrel{\simeq}{\to} b\} on the standard interval object – which is (by the definition of natural transformation) the action groupoid

BG I=G\\G//G \mathbf{B}G^I = G\backslash \backslash G // G

for the action of GG on itself, by inverse left and direct right multiplication separately: the naturality square of a natural transformation defining a morphism g→h 1,h 2h 1 −1gh 2g \stackrel{h_1,h_2}{\to} h_1^{-1} g h_2 in this groupoid is the commuting square

• →g • h 1↓ ↓ h 2 • →h 1 −1gh 2 • \array{ \bullet &\stackrel{g}{\to}& \bullet \\ {}^{\mathllap{h_1}}\downarrow && \downarrow^{\mathrlap{h_2}} \\ \bullet &\stackrel{h_1^{-1}g h_2}{\to}& \bullet }

in BG=*//G\mathbf{B}G = {*}//G.

The pullback of the top right corner of the above defining limit diagram is

(G\\G×G\\G)//G → BG ↓ ↓ (Id,Id) (G\\G//G)×(G\\G//G) → BG×BG \array{ (G\backslash\backslash G \times G\backslash\backslash G)//G &\to& \mathbf{B}G \\ \downarrow && \downarrow^{\mathrlap{(Id,Id)}} \\ (G\backslash\backslash G//G) \times (G\backslash\backslash G//G) &\to& \mathbf{B}G \times \mathbf{B}G }

identifying the two actions from the right, and then the remaining pullback completing the limit diagram is

G\\(G×G)//G → (G\\G×G\\G)//G ↓ ↓ BG →(Id,Id) BG×BG \array{ G\backslash\backslash (G\times G)//G &\to& (G\backslash\backslash G \times G\backslash\backslash G)//G \\ \downarrow && \downarrow \\ \mathbf{B}G &\stackrel{(Id,Id)}{\to}& \mathbf{B}G \times \mathbf{B}G }

now identifying also the two actions from the left, so that G\\(G×G)//GG\backslash\backslash (G\times G)//G is the action groupoid of GG acting diagonally on G×GG \times G by multiplication from the left and from the right, separately.

To see better what this is, we pass to an equivalent smaller groupoid (the homotopy pullback is defined, of course, only up to weak equivalence). Notice that every morphism (g 1,g 2)→h 1,h 2(g′ 1,g′ 2)(g_1,g_2) \stackrel{h_1,h_2}{\to} (g'_1, g'_2) in G\\(G×G)//GG\backslash\backslash (G\times G)//G corresponding to a natural transformation

• →(g 1,g 2) • (h 1,h 1)↓ ↓ (h 2,h 2) • →(h 1 −1g 1h 2,h 1 −1g 2h 2) • \array{ \bullet &\stackrel{(g_1,g_2)}{\to}& \bullet \\ {}^{\mathllap{(h_1,h_1)}}\downarrow && \downarrow^{\mathrlap{(h_2,h_2)}} \\ \bullet &\stackrel{(h_1^{-1} g_1 h_2, h_1^{-1} g_2 h_2)}{\to} & \bullet }

between functors I×I→BG×BGI\times I \to \mathbf{B}G \times \mathbf{B}G may always be decomposed as

• →(g 1,g 2) • (e,e)↓ ↓ (g 2 −1,g 2 −1) • →(g 1g 2 −1,e) • (h 1,h 1)↓ ↓ (h 1,h 1) • →(h 1 −1(g 1g 2 −1)h 1,e) • (e,e)↓ ↓ (g′ 2,g′ 2) • →(h 1 −1(g 1g 2 −1)h 1g′ 2,g′ 2) •. \array{ \bullet &\stackrel{(g_1,g_2)}{\to}& \bullet \\ {}^{\mathllap{(e,e)}}\downarrow && \downarrow^{\mathrlap{(g_2^{-1}, g_2^{-1})}} \\ \bullet &\stackrel{(g_1 g_2^{-1}, e)}{\to}& \bullet \\ {}^{\mathllap{(h_1,h_1)}}\downarrow && \downarrow^{\mathrlap{(h_1,h_1)}} \\ \bullet &\stackrel{(h_1^{-1}(g_1 g_2^{-1})h_1, e)}{\to}& \bullet \\ {}^{\mathllap{(e,e)}}\downarrow && \downarrow^{\mathrlap{(g'_2,g'_2)}} \\ \bullet &\stackrel{(h_1^{-1}(g_1 g_2^{-1})h_1 g'_2, g'_2)}{\to}& \bullet } \,.

Staring at this for a moment shows that this is a unique factorization of every morphism through one of the form

• →(g,e) • k↓ ↓ k • →(Ad kg,e) •, \array{ \bullet & \stackrel{(g,e)}{\to} & \bullet \\ {}^{\mathllap{k}}\downarrow && \downarrow^{\mathrlap{k}} \\ \bullet & \stackrel{(Ad_k g,e)}{\to} & \bullet } \,,

which is naturally identified with a morphism in the action groupoid G// AdGG//_{Ad} G of the adjoint action of GG on itself.

This means that the inclusion

G// AdG↪G\\(G×G)//G G//_{Ad} G \stackrel{}{\hookrightarrow} G\backslash\backslash(G \times G)//G

given by this identification is essentially surjective and full and faithful, and hence an equivalence of groupoids.

So in conclusion we have that the free loop space object of the delooping BG\mathbf{B}G of a group is

ℒBG≃G// AdG. \mathcal{L} \mathbf{B}G \simeq G//_{Ad}G \,.

Chern character

We describe how the Chern character of vector bundles over XX may be realized in terms of the cohomology of the free loop space object ℒX\mathcal{L}X.

Assume now CC is a nice category of smooth spaces, and let XX be an object of CC.

Consider a group object GG in CC and a representation of GG given by a group homomorphism to the general linear group (in CC): ρ:G→GL(n;ℂ)\rho:G\to GL(n;\mathbb{C}). For instance GG could be GL(n)GL(n) itself and this morphism the identity.

The trace of the representation ρ\rho is invariant under conjugation in the group and so defnes a map Tr(ρ):G// AdG→ℂTr(\rho): G//_{Ad}G\to \mathbb {C} – a class function. By the equivalence ℒBG≃G// AdG\mathcal{L}\mathbf{B}G \simeq G//_{Ad} G discussed above, this may be regarded as a characteristic class

Tr(ρ(−)):ℒBG→ℂ Tr(\rho(-)) : \mathcal{L}\mathbf{B}G\to \mathbb {C}

on the free loop space of BG\mathbf{B}G.

The cocycle g:X→BGg : X\to \mathbf{B}G of a GG-principal bundle on XX transgresses to a cocycle

ℒg:ℒX→ℒBG \mathcal{L} g : \mathcal{L}X \to \mathcal{L}\mathbf{B}G

on the free loop space, by the functoriality of the free loop space object construction.

The above characteristic class of this cocycle is the composite morphism

Tr(ρ(ℒg)):ℒX→ℒBG→ℂ, Tr(\rho(\mathcal{L}g)) : \mathcal{L}X \to \mathcal{L} \mathbf{B}G \to \mathbb{C} \,,

which by the AdAd-invariance of the trace is now S 1S^1-invariant and hence defines an element in the cyclic cohomology C(ℒX,ℂ) S C 1C(\mathcal{L}X,\mathbb{C})^{S^1_C} of XX.

The Hom-space C(ℒX,ℂ)C(\mathcal{L}X,\mathbb{C}) is a model for the graded commutative algebra of complex-valued differential forms on XX, with the categorical circle action corresponding to the de Rham differential. Hence C(ℒX,ℂ) S C 1C(\mathcal{L}X,\mathbb{C})^{S^1_C} is a model for closed forms and maps to de Rham cohomology H dR •(X)H_{dR}^\bullet(X) of XX. If the de Rham theorem holds for XX in CC, then this may be identified with the real cohomology H •(X,ℝ)H^\bullet(X,\mathbb{R}).

In the case that G=GL(∞;ℂ)G=GL(\infty;\mathbb{C}), the compatibility of the trace with direct sums and tensor products of vector bundles over XX makes the above construction a ring homomorphism K(X)→H dR(X)K(X)\to H_{dR}(X) from the topological K-theory of XX to de Rham cohomology, hence a very good candidate to being the Chern character
