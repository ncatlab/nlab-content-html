Modal type theory

A modal type theory for bireflection modalities has been proposed in RFL21.

We show the inference rules which are added to plain intuitionistic type theory for encoding the presence of a bireflection classical modality ‚ôÆ\natural on the type system.

(Beware the notational asymmetry: An underline of a context denotes application of ‚ôÆ\natural, but an underline of a type in context only means its pullback along the counit œµ ‚ôÆ\epsilon^{\natural}. The idea is that underlines indicate dependency on ‚ôÆ\natural-modal variables, to be distinguished of a type itself being ‚ôÆ\natural-modal or not.)
