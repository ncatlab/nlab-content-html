Idea

An Archimedean ordered field is an ordered field that satisfies the archimedean property.

Definition

The rational numbers are the initial ordered field, so for every ordered field FF there is a field homomorphism h:ℚ→Fh:\mathbb{Q}\to F. FF is Archimedean if for all elements a∈Fa\in F and b∈Fb\in F, if a<ba \lt b, then there exists a rational number q∈ℚq\in \mathbb{Q} such that a<h(q)a \lt h(q) and h(q)<bh(q) \lt b.

Properties

Every Archimedean ordered field is a dense linear order. This means that the Dedekind completion of every Archimedean ordered field is the field of real numbers.

Every Archimedean ordered field is a differentiable space:

Pointwise continuous functions

Let FF be an Archimedean ordered field. A function f:F→Ff:F \to F is continuous at a point c∈Fc \in F if

ff is pointwise continuous in FF if it is continuous at all points cc:

isPointwiseContinuous(f)≔∀c∈F.isContinuousAt(f,c)isPointwiseContinuous(f) \coloneqq \forall c \in F. isContinuousAt(f, c)

The set of all pointwise continuous functions is defined as

C 0(F)≔{f∈F→F|isPointwiseContinuous(f)}C^0(F) \coloneqq \{f \in F \to F \vert isPointwiseContinuous(f)\}

Examples

Archimedean ordered fields include

  • rational numbers
  • real closed fields
  • real numbers

Non-Archimedean ordered fields include

  • finite fields

  • complex numbers

  • p-adic numbers

  • surreal numbers
