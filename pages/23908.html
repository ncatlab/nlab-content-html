Idea

In type theory, a 1-truncated A 4 A_4 -space. In category theory, a magmoidal groupoid with unit with an associator that satisfy the pentagon identities but no higher coherence laws for the unitors.

Definitions

In type theory

In intensional type theory with function extensionality, a groupoid is a 1-truncated type and an A 4A_4-spatial groupoid is a groupoid that is also an A 4 A_4 -space.

In category theory

In category theory, an A 4A_4-spatial groupoid is a magmoidal groupoid with unit (G,⊗,I,λ (−),ρ (−))(G, \otimes, I, \lambda_{(-)}, \rho_{(-)}) with natural unitary ismorphisms α A,B,C:A⊗(B⊗C)≃ †(A⊗B)⊗C\alpha_{A, B, C} \colon A \otimes (B \otimes C) \simeq^\dagger (A \otimes B) \otimes C such that the pentagon identity is satisfied for all objects AA, BB, CC, and DD:

α A,B,C⊗D∘α A⊗B,C,D=(id A⊗α B,C,D)∘α A,B⊗C,D∘(α A,B,C⊗id D)\alpha_{A,B,C \otimes D} \circ \alpha_{A \otimes B, C, D} = (id_A \otimes \alpha_{B,C,D}) \circ \alpha_{A,B \otimes C, D} \circ (\alpha_{A,B,C} \otimes id_D)
