Idea

A locally presentable category is a category which contains a small set SS of small objects such that every object is a nice colimit over objects in this set.

This says equivalently that a locally presentable category ùíû\mathcal{C} is a reflective localization ùíû‚Ü™PSh(S)\mathcal{C} \hookrightarrow PSh(S) of a category of presheaves over SS. Since here PSh(S)PSh(S) is the free colimit completion of SS and the localization imposes relations, this is a presentation of ùíû\mathcal{C} by generators and relations, hence the name (locally) presentable category.

See also at locally presentable categories - introduction.

Definition

There are many equivalent characterizations of locally presentable categories. The following is one of the most intuitive, equivalent characterizations are discussed below.

Definition

(locally presentable category)

A category ùíû\mathcal{C} is called locally presentable if

  1. it is an accessible category;

  2. it has all small colimits.

This means

  1. ùíû\mathcal{C} is a locally small category;

  2. ùíû\mathcal{C} has all small colimits;

  3. there exists a small set S‚Ü™Obj(ùíû)S \hookrightarrow Obj(\mathcal{C}) of Œª\lambda-compact objects that generates ùíû\mathcal{C} under Œª\lambda-filtered colimits for some regular cardinal Œª\lambda.

Remark

If follows that every object in a locally presentable category is a small object.

Remark

The locally in locally presentable category refers to the fact that it is the objects of (in) the category that are presentable, not the category as such (which is itself an object of Cat). This permits the distinction between, for instance, locally presentable categories, and finitely presentable objects of Cat, which could be called ‚Äúfinitely presentable categories‚Äù. In practice, however, it is common to drop ‚Äúlocally‚Äù from ‚Äúlocally presentable category‚Äù without modifying the meaning.

Remark

Since a small object is one which is Œ∫\kappa-compact for some Œ∫\kappa, and any Œ∫\kappa-compact object is also Œª\lambda-compact for any Œª>Œ∫\lambda \gt \kappa, it follows that there exists some Œ∫\kappa such that every object of the colimit-generating set SS is Œ∫\kappa-compact.

This provides a ‚Äústratification‚Äù of the class of locally presentable categories, as follows.

Definition

(locally Œ∫\kappa-presentable category)

For Œ∫\kappa a regular cardinal, a locally Œ∫\kappa-presentable category is a locally presentable category, def. , such that the colimit-generating set SS may be taken to consist of Œ∫\kappa-compact objects.

Remark

Thus, a locally presentable category is one which is locally Œ∫\kappa-presentable for some regular cardinal Œ∫\kappa (hence also for every Œª>Œ∫\lambda\gt\kappa). In fact, in this case the fourth condition is redundant; once we know that there is a colimit-generating set consisting of Œ∫\kappa-compact objects, it follows automatically that every object is Œª\lambda-compact for some Œª\lambda (though there is no uniform upper bound on the required size of Œª\lambda). Moreover, colimit-generation is also stronger than necessary; it suffices to have a strong generator consisting of small objects.

A locally ‚Ñµ 0{\aleph}_0-presentable category is called a locally finitely presentable category.

Properties

Equivalent characterizations

There are various equivalent characterizations of locally presentable categories.

As limit-preserving functor categories

Proposition

(as limit sketches)

Locally presentable categories are precisely the categories of models of limit-sketches.

This is Ad√°mek & Rosick√Ω (1994), corollary 1.52.

Restricted to locally finitely presentable categories this becomes:

Locally finitely presentable categories, def. , are equivalently the categories of finite limit preserving functors C‚ÜíSetC \to Set, for small finitely complete categories CC.

For the more detailed statement see below at Gabriel-Ulmer duality. Equivalently this says that:

Locally finitely presentable categories are equivalently models of finitary essentially algebraic theories.

As localizations of presheaf categories

Locally presentable categories are precisely the accessibly embedded full reflective subcategories

Remark

Here accessibly embedded means that C‚Ü™Psh(K)C \hookrightarrow Psh(K) is an accessible functor, which in turn means that CC is closed in Psh(K)Psh(K) under Œ∫\kappa-filtered colimits for some regular cardinal Œ∫\kappa.

See also at sheaf toposes are equivalently the left exact reflective subcategories of presheaf toposes.

Corollary

Locally presentable categories are complete.

Proof

A reflective subcategory of a complete category is complete, since monadic functors reflect limits, and the above proposition shows that any locally presentable category is a reflective subcategory of a presheaf category, which is complete.

Well-poweredness and well-copoweredness

  • Every locally presentable category is well-powered, since it is a full reflective subcategory of a presheaf topos, so its subobject lattices are subsets of those of the latter.

  • Every locally presentable category is also well-copowered. This is shown in Ad√°mek & Rosick√Ω (1994), Prop. 1.57 & Thm. 2.49.

Examples and applications

Locally finitely presentable categories

We list examples of locally finitely presentable categories, def. .

Example

The category Set of sets is locally finitely presentable.

For notice that every set is the directed colimit over the poset of all its finite subsets.

Moreover, a set S‚ààSetS \in Set is a Œ∫\kappa-compact object precisely if it has cardinality |S|<Œ∫|S| \lt \kappa. So all finite sets are ?0_0-compact.

Hence a a set of generators that exhibits SetSet as a locally finitely complete category is given by the set containing one finite set of cardinality n‚àà‚Ñïn \in \mathbb{N} for all nn.

Example

More generally, for CC any small category the category of presheaves Set CSet^C is locally finitely presentable.

This follows with Gabriel-Ulmer duality: the finite limit completion of CC, Lex(C)Lex(C), is also small, and Set CSet^C is equivalent to the category of finitely continuous functors Lex(C)‚ÜíSetLex(C) \to Set.

Example

More generally still, if AA is locally finitely presentable and CC is small, then A CA^C is locally finitely presentable.

To see this, embed AA as a finitely-accessible reflective subcategory of a presheaf topos Set BSet^B, and then note that by 2-functoriality of (‚àí) C(-)^C we get A CA^C as a finitely-accessible reflective subcategory of Set B√óCSet^{B \times C}.

Example

The category of algebras of a Lawvere theory, for example Grp, is locally finitely presentable. A TT-algebra AA is finitely presented if and only if the hom-functor Alg T(A,‚àí)Alg_T(A, -) preserves filtered colimits, and any TT-algebra can be expressed as a filtered colimit of finitely presented algebras.

Example

The category of coalgebras over a field kk is locally finitely presentable; similarly the category of commutative coalgebras over kk is locally finitely presentable.

Example

A poset, regarded as a category, is locally finitely presentable if it is a complete lattice which is algebraic (each element is a directed join of finite elements).

Remark

(counter-examples)

  • The category FinSet of finite sets is not locally finitely presentable, as it does not have all countable colimits.

  • The category of fields and field homomorphisms is not locally presentable, as it does not have all binary coproducts (for instance, there are none between fields of differing characteristics).

  • TopologicalSpaces is not locally finitely presentable.

  • The opposite category of a locally presentable category (in particular, a locally finitely presentable category) is never locally presentable, unless it is a poset. This is Gabriel-Ulmer, Satz 7.13.

Locally presentable categories

Example

A poset, considered as a category, is locally presentable precisely if it is a complete lattice.

Example

Every Grothendieck abelian category is locally presentable [Beke (200), Prop. 3.10, cf. Krause (2015), Cor. 5.2].

This implies in particular (by this example at Grothendieck abelian category) that for RR a commutative ring (internal to any Grothendieck topos):

  • the category RRMod of RR-modules is locally presentable

Combinatorial model categories

A combinatorial model category is a model category that is in particular a locally presentable category.

Orthogonal subcategory problem

Given a class of morphisms Œ£\Sigma in a locally presentable category, the answer to the orthogonal subcategory problem for Œ£ ‚ä•\Sigma^\perp is affirmative if Œ£\Sigma is small, and is affirmative for any class Œ£\Sigma assuming the large cardinal axiom known as Vopenka's principle.
