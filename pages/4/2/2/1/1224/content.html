Idea

A functor F:C‚ÜíDF : C \to D is final, if we can restrict diagrams on DD to diagrams on CC along FF without changing their colimit.

Dually, a functor is initial if pulling back diagrams along it does not change the limits of these diagrams.

Beware that this property is pretty much unrelated to that of a functor being an initial object or terminal object in the functor category [C,D][C,D]. The terminology comes instead from the fact that an object d‚ààDd\in D is initial (resp. terminal) just when the corresponding functor d:1‚ÜíDd:1\to D is initial (resp. final).

Remark

(warning on terminology)
In older references (and also some others like HTT), final functors are sometimes called cofinal, the terminology having been imported from order theory (e.g. cofinality). However, this is confusing in category theory because usually the prefix ‚Äúco-‚Äù denotes dualization. In at least one place (Borceux) this non-dualization was treated as a dualization and the word ‚Äúfinal‚Äù used for the dual concept, but in general it seems that the consensus is to use ‚Äúfinal‚Äù for what used to be called ‚Äúcofinal‚Äù, and ‚Äúinitial‚Äù for the dual concept (since ‚Äúco-final‚Äù would be ambiguous). For example, Johnstone in Sketches of an Elephant before Proposition B2.5.12 says: : Traditionally, final functors were called ‚Äòcofinal functors‚Äô; but this use of ‚Äòco‚Äô is potentially misleading as it has nothing to do with dualization ‚Äî it is derived from the Latin ‚Äòcum‚Äô rather than ‚Äòcontra‚Äô ‚Äî and so it is now generally omitted.

See also the warning at final ( ‚àû , 1 ) (\infty,1) -functor (here).

Definition

Definition

A functor F:C‚ÜíDF : C \to D is final if for every object d‚ààDd \in D the comma category (d/F)(d/F) is (non-empty and) connected (the non-emptiness condition is redundant since connected categories are non-empty by convention).

A functor F:C‚ÜíDF : C \to D is initial if the opposite F op:C op‚ÜíD opF^{op} : C^{op} \to D^{op} is final, i.e. if for every object d‚ààDd \in D the comma category (F/d)(F/d) is connected.

Examples

Example

(inclusion of a terminal object is final functor)
If DD has a terminal object then the functor F:*‚ÜíDF : {*} \to D that picks that terminal object is final: for every d‚ààDd \in D the comma category d/Fd/F is equivalent to **. The converse is also true: if a functor *‚ÜíD*\to D is final, then its image is a terminal object.

In this case the statement about preservation of colimits states that the colimit over a category with a terminal object is the value of the diagram at that object. Which is also readily checked directly.
