¬†Idea

The boolean domain in dependent type theory.

Definition

The type of booleans, denoted Bool\mathrm{Bool}, ùüö\mathbb{2}, or 2\mathbf{2}, can be defined in a dependent type theory in many different ways:

  • as a positive inductive type representing the homotopy-initial type with two elements.

  • as the type of decidable propositions, itself defined in multiple different ways

  • as the 0-sphere type, the suspension type of the empty type

As the type of decidable propositions

The type of booleans can also be defined as the type of all decidable propositions, provided one already has disjunctions and the negations defined in the type theory:

  • using the type of all propositions, as the subuniverse of decidable propositions.

  • as a homotopy-terminal univalent type family of decidable propositions, suitably defined.

  • as a record type with type fields, with inference rules that mimic that of the negative dependent sum type ‚àë P:UisProp(P)√óisDecidable(P)\sum_{P:U} \mathrm{isProp}(P) \times \mathrm{isDecidable}(P) or ‚àë P:UisProp(T(P))√óisDecidable(T(P))\sum_{P:U} \mathrm{isProp}(T(P)) \times \mathrm{isDecidable}(T(P)) respectively, but for all types, not just the UU-small types.

Using the type of all propositions

Suppose that we have a type of propositions Prop\mathrm{Prop}. Then, the type of booleans is defined as

Bool‚â°‚àë P:PropP‚à®¬¨P\mathrm{Bool} \equiv \sum_{P:\mathrm{Prop}} P \vee \neg P

As a homotopy-terminal type

A univalent family of decidable propositions consists of a type AA and a type family (B(x)) x:A(B(x))_{x:A} such that

As a record type with type fields

The type of booleans Bool\mathrm{Bool} is a record type where one of the fields is a type, consisting of

  • a type AtypeA \; \mathrm{type}

  • a witness p:isProp(A)√ó(A‚à®¬¨A)p:\mathrm{isProp}(A) \times (A \vee \neg A) that AA is a decidable proposition.

As a result, the type of booleans behaves as a type universe.

Descent and large recursion

The descent for the type of booleans states that given any types AA and BB one can construct a type family x:ùüö‚ä¢descFam ùüö A,B(x)x:\mathbb{2} \vdash \mathrm{descFam}_\mathbb{2}^{A, B}(x) with equivalences of types

Large recursion for the type of booleans strengthens the equivalences of types in descent to judgmental equality of types

descFam ùüö A,B(0)‚â°AanddescFam ùüö A,B(1)‚â°B\mathrm{descFam}_\mathbb{2}^{A, B}(0) \equiv A \quad \mathrm{and} \quad \mathrm{descFam}_\mathbb{2}^{A, B}(1) \equiv B

Extensionality principle of the type of booleans

The elements of the type of booleans represent certain truth values or propositions, namely, true and false. By the principle of propositions as some types, truth values or propositions are represented as certain types: specifically, true or 11 is represented by the unit type ùüô\mathbb{1}, and false or 00 is represented by the empty type ùüò\mathbb{0}. The type of booleans is a Tarski universe through the following type family:

x:Bool‚ä¢El(x)‚âî(x= Bool1)√ó¬¨(x= Bool0)x:\mathrm{Bool} \vdash \mathrm{El}(x) \coloneqq (x =_\mathrm{Bool} 1) \times \neg (x =_\mathrm{Bool} 0)

The extensionality principle of the type of booleans is then given by the univalence axiom:

ŒìctxŒì,x:Bool,y:Bool‚ä¢ua(x,y):Id Bool(x,y)‚âÉ(El(x)‚âÉEl(y))\frac{\Gamma \; \mathrm{ctx}}{\Gamma, x:\mathrm{Bool}, y:\mathrm{Bool} \vdash \mathrm{ua}(x, y):\mathrm{Id}_\mathrm{Bool}(x, y) \simeq (\mathrm{El}(x) \simeq \mathrm{El}(y))}

Since the empty type is not equivalent to the unit type, this automatically implies that 00 is not equal to 11. The extensionality principle of the type of booleans can be proven from descent of the type of booleans, see Sattler 2023 for a proof.

Relation to sum types

The sum types can be defined in terms of the type of booleans and the dependent sum type. Given types AA and BB, the sum type A+BA + B is defined as

Other properties

  • The type of booleans is the suspension type of the empty type, and the suspension of the type of booleans is the circle type. Geometrically, the type of booleans is a zero-dimensional sphere.
