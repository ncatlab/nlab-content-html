Idea

An infinitesimal interval object is like an interval object that is an infinitesimal space.

Where maps out of an interval object model paths that may arrange themselves into path ‚àû-groupoids, maps out of infinitesimal interval object model infinitesimal paths that may arrange themselves into infinitesimal path ‚àû-groupoid.

Definition

In any lined topos (ùíØ,(R,+,‚ãÖ))(\mathcal{T},(R,+,\cdot)) the line object RR is naturally regarded as a cartesian interval object.

R 0 *‚Üó ‚Üñ 1 * * *. \array{ && R \\ & {}^{0_{*}}\nearrow && \nwarrow^{1_{*}} \\ {*} &&&& {*} } \,.

If the lined topos (ùíØ,R)(\mathcal{T}, R) is also a smooth topos in that it satisfies the Kock-Lawvere axiom it makes sense to consider the subobject DD of RR defined as the equalizer

D=lim(R‚Üí0‚Üí(‚àí) 2R) D = \lim\left( R \stackrel{\stackrel{(-)^2}{\to}}{\underset{0}{\to}} R \right)

or equivalently expressed in topos logic as

D={x‚ààR|x 2=0}. D = \{x \in R | x^2 = 0 \} \,.

and think of DD as the infinitesimal interval object of the smooth topos (ùíØ,R)(\mathcal{T},R) inside its finite interval object RR. By the axioms satisfied by a smooth topos it is in particular an infinitesimal object.

Various constructions induced by a finite interval object have their infinitesimal analog for infinitesimal interval objects.

Introduction

Recall the discussion at interval object of how the interval *‚äî*‚Üí‚ÜíR{*}\sqcup {*} \stackrel{\to}{\to} R in ùíØ\mathcal{T} alone gave rise to the cosimplicial object

We want to mimic that construction with the finite interval RR replaced by the infinitesimal interval object DD, to get a simplicial object Œ† inf(X)\Pi^{inf}(X) for every object XX.

While the infinitesimal situation is formally very similar to the finite situation, one technical diference is that the infinitesimal interval does not fit into a nontrivial cospan as the finite interval does. This is because DD typically has a unique morphism *‚ÜíD{*} \to D from the terminal object, as a consequence of the fact that all the infinitesimal elements it contains are genuinely generalized elements.
