Using the cellular cohomology of complex projective space

There is a proof using that the cellular cohomology of complex projective space ℂℙ n−1\mathbb{CP}^{n-1} is (see there)

H i(ℂℙ n−1,ℤ)≅ℤ a i, H^i(\mathbb{CP}^{n-1}, \mathbb{Z}) \cong \mathbb{Z}^{a_i} \mathrlap{\,,}

where

a i={1 ifi∈{2j|0≤j≤n−1} 0 otherwise a_i = \left\{ \begin{array}{ll} 1 & \text{if}\; i \in \{ 2j \vert 0 \leq j \leq n-1 \} \\ 0 & \text{otherwise} \end{array} \right.

and in which the cup product induces a ring-isomorphism

H *(ℂℙ n,ℤ)≃ℤ[c]/c nℤ[c] H^* (\mathbb{CP}^n, \mathbb{Z}) \simeq \mathbb{Z}[c] / c^n \mathbb{Z}[c]

where cc is the integral first Chern class (see here

Theorem

Let KK be a finite dimensional vector space over ℂ\mathbb{C} with μ:K⊗ ℂK→K\mu \colon K \otimes_{\mathbb{C}} K \rightarrow K a ℂ\mathbb{C}-linear map and an element designated by η:ℂ→K\eta : \mathbb{C} \rightarrow K satisfying the following two properties:

  • (unitality) μ∘(η⊗Id K)∘η 1=Id K=μ∘(Id K⊗η)∘η 2\mu \circ \left( \eta \otimes Id_{K} \right) \circ \eta_1 \ = \ Id_K \ = \ \mu \circ \left( Id_K \otimes \eta \right) \circ \eta_2 where η 1:{*}×ℂℙ n−1→ℂℙ n−1\eta_1 : \{ * \} \times \mathbb{CP}^{n-1} \rightarrow \mathbb{CP}^{n-1} and η 2:ℂℙ n−1×{*}→ℂℙ n−1\eta_2 : \mathbb{CP}^{n-1} \times \{ * \} \rightarrow \mathbb{CP}^{n-1} are the unit isomorphisms with respect to the cartesian closed category induced by product.

  • (no zero-divisors) for a,b∈Ka, b \in K, if μ(a⊗b)=0\mu (a \otimes b) = 0, then either a=0a = 0 or b=0b = 0.

Then KK is isomorphic to ℂ\mathbb{C} itself as a ℂ \mathbb{C} -vector space, i.e. dim(K)=1\text{dim}(K) = 1.

Note that we obtain such a nonassociative ℂ\mathbb{C}-algebra as the splitting field of any complex irreducible polynomial.

Proof

Using the second property we obtain a continuous function

μ| ℂ n−{0}:(ℂ n−{0})×(ℂ n−{0})→ℂ n−{0} \mu |_{ \mathbb{C}^n - \{ 0 \} } \ : \ \left( \mathbb{C}^{n} - \{ 0 \} \right) \times \left( \mathbb{C}^{n} - \{ 0 \} \right)\ \rightarrow\ \mathbb{C}^{n} - \{ 0 \}

Bilinearity implies that π∘μ| ℂ n−{0}\pi \circ \mu|_{ \mathbb{C}^n - \{ 0 \} } factors through the product ℂℙ n−1×ℂℙ n−1\mathbb{CP}^{n-1} \times \mathbb{CP}^{n-1} where π:ℂ n−{0}→ℂℙ n−1\pi : \mathbb{C}^{n} - \{ 0 \} \rightarrow \mathbb{CP}^{n-1} is the quotient map. We get a continuous function

[μ| ℂ n]:ℂℙ n−1×ℂℙ n−1→ℂℙ n−1 [ \mu|_{ \mathbb{C}^n} ] \ : \ \mathbb{CP}^{n-1} \times \mathbb{CP}^{n-1} \rightarrow \mathbb{CP}^{n-1}

We obtain two equations by applying degree-2 integral cohomology to the two unit laws in the first property, irrespective of any associativity or commutativity of KK:

H 2(μ∘(η×Id ℂℙ n−1),ℤ)=Id H 2(ℂℙ n−1,ℤ)=H 2(μ∘(Id ℂℙ n−1×η),ℤ). H^2 \left( \mu \circ \left( \eta \times Id_{\mathbb{CP}^{n-1}} \right) , \mathbb{Z} \right) = Id_{H^2 \left( \mathbb{CP}^{n-1} , \mathbb{Z} \right) } = H^2 \left( \mu \circ \left( Id_{\mathbb{CP}^{n-1}} \times \eta \right) , \mathbb{Z} \right) \mathrlap{\,.}

Let y∈H 2(ℂℙ n−1)y \in H^2(\mathbb{CP}^{n-1}) be the generator of the cohomology ring H *(ℂℙ n−1)H^* ( \mathbb{CP}^{n-1} ). We have:

H 2(ℂℙ n−1×ℂℙ n−1;ℤ)≅H 2(ℂℙ n−1;ℤ)⊕H 2(ℂℙ n−1;ℤ). H^2\big( \mathbb{CP}^{n-1} \times \mathbb{CP}^{n-1}; \mathbb{Z} \big) \cong H^2\big( \mathbb{CP}^{n-1}; \mathbb{Z} \big) \oplus H^2\big( \mathbb{CP}^{n-1}; \mathbb{Z} \big) \mathrlap{\,.}

We can write

H 2(μ,ℤ)(y)=a⋅(1⊗y)+b⋅(y⊗1) H^2(\mu,\mathbb{Z})(y) = a \cdot (1 \otimes y) + b \cdot (y \otimes 1)

for some a,b∈ℤa, b \in \mathbb{Z}.

The two equations obtained from the unit laws for KK imply that:

a=1,b=1. a = 1, \, b = 1 \mathrlap{\,.}

There is an isomorphism of Hopf algebras in integral cohomology of

(ℤ[x]/x nℤ[x])⊗ ℤ(ℤ[x]/x nℤ[x])≅H *(ℂℙ n−1;ℤ)⊗ ℤH *(ℂℙ n−1;ℤ)≅H *(ℂℙ n−1×ℂℙ n−1;ℤ). \big( \mathbb{Z}[x]/x^n\mathbb{Z}[x] \big) \otimes_{\mathbb{Z}} \big( \mathbb{Z}[x]/x^n\mathbb{Z}[x] \big) \cong H^*\big( \mathbb{CP}^{n-1}; \mathbb{Z} \big) \otimes_{\mathbb{Z}} H^*\big( \mathbb{CP}^{n-1}; \mathbb{Z} \big) \cong H^*\big( \mathbb{CP}^{n-1} \times \mathbb{CP}^{n-1}; \mathbb{Z} \big) \mathrlap{\,.}

But H *(μ,ℤ)H^*(\mu,\mathbb{Z}) is a ring homomorphism, so

H *(μ,ℤ)(y n)=(1⊗y+y⊗1) n=∑ i=0 n(ni)(1⊗y) i⋅(y⊗1) n−i=∑ i=0 ny n−i⊗y i H^*(\mu,\mathbb{Z})(y^n) = (1 \otimes y + y \otimes 1)^n = \sum_{i = 0}^n {\binom{n}{i}} (1 \otimes y)^i \cdot (y \otimes 1)^{n-i} = \sum_{i = 0}^n y^{n-i} \otimes y^{i}

as elements of the commutative Hopf-algebra

H *(ℂℙ n−1,ℤ)⊗ ℤH *(ℂℙ n−1,ℤ)≅(ℤ[x]/x nℤ[x])⊗(ℤ[x]/x nℤ[x]). H^*(\mathbb{CP}^{n-1},\mathbb{Z}) \otimes_{\mathbb{Z}} H^*(\mathbb{CP}^{n-1},\mathbb{Z}) \;\cong\; \left( \mathbb{Z}[x]/x^n \mathbb{Z}[x] \right) \otimes \left( \mathbb{Z}[x]/x^n \mathbb{Z}[x] \right) \mathrlap{\,.}

The left-hand side is 00.

The right hand side is

∑ i=1 n−1y n−i⊗y i. \sum_{i = 1}^{n-1} y^{n-i} \otimes y^{i} \mathrlap{\,.}

By way of contradiction, suppose that n=dim(K)n = \text{dim}(K) is greater than 11. In this case, each of the terms y i⊗y n−iy^i \otimes y^{n-i} is nonzero for 0<i<n0 \lt i \lt n, and their sum is as well.

This implies that

∑ i=1 n−1(ni)y n−i⊗y i \sum_{i= 1}^{n-1} {\binom{n}{i}} y^{n-i} \otimes y^{i}

is not zero, which contradicts that y ny^n is zero using the above equation.

This implies n=1n = 1 and hence the claim.
