Karoubi 1978 §III.4 defined topological K-theory classes equivalently by Clifford module bundles, where a Cl nCl_n-module represents a class in K nK^n and represents the trivial class if, roughly, it extends to a Cl n+1Cl_{n+1}-module.

(Over the point this is the Atiyah-Bott-Shapiro isomorphism.)

Idea

We have a sequence of Clifford algebras Cl nCl_n which are generated by nn anticommuting square roots of ±1\pm 1. The sequence is periodic up to Morita equivalence; Cl 8Cl_8 is ℝ(16)\mathbb{R}(16), the algebra of 16×1616 \times 16 real matrices, which is Morita equivalent to ℝ\mathbb{R}, and from then on it repeats every 8 with extra matrix dimensions thrown in – this is Bott periodicity. (Notice that here we treat Clifford algebras as ℤ 2\mathbb{Z}_2-graded algebras: while Cl 6Cl_6 is Morita equivalent to ℝ(8)\mathbb{R}(8) as an algebra, it is not so as a graded algebra.)

It turns out that K n(X)K^n(X) can be represented geometrically by Clifford module bundles over XX. Start with K 0K^0; we know that elements of K 0(X)K^0(X) are ‘formal differences’ V−WV - W of vector bundles over XX (virtual vector bundles). We can model the formal difference V−WV - W with an honest geometric object by using the ℤ 2\mathbb{Z}_2-graded vector bundle V⊕WV \oplus W, where VV is even and WW is odd. Such a thing should represent the zero class in K-theory just when VV and WW are isomorphic; this can be rephrased as saying that there exists an odd operator ee on V⊕WV \oplus W (hence, taking VV to WW and vice versa) such that e 2=1e^2 = 1. But this just says that V⊕WV \oplus W has an action of the first Clifford algebra Cl 1Cl_1.

The idea is now to generalize this to K-theory in other degrees.

Definition

By default, let the ground field be the complex numbers.

Consider

  • XX a suitable topological space,

  • Vec X Vec_X ? the category of topological vector bundles over XX,

  • p,q∈ℕp,q \in \mathbb{N},

  • Cl p,qCl^{p,q} the Clifford algebra with that signature,

  • Cl p,qMod X≔Cl p,qMod(Vec X)Cl^{p,q}Mod_X \,\coloneqq\, Cl^{p,q}Mod\big(Vec_X\big) the category of module objects,

  • Cl X p,q+1→Cl X p,qCl^{p,q+1}_X \xrightarrow{\;} Cl^{p,q}_X the forgetful functor.

Then Karoubi 1978 Def. 4.11 defines the K-theory group K p,q(X)K^{p,q}(X) to be the Grothendieck group of this forgetful functor.

According to Karoubi 78 Thm. 4.12 this definition reproduces the ordinary definition of topological K-theory groups in degree p−qp-q.
