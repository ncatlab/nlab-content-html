Donsker–Varadhan nonequilibrium correction

Idea

Given an irreducible continuous-time Markov chain at stationarity, the Hessian of the Donsker–Varadhan rate function? splits into a detailed-balance part and a remainder. The remainder — the nonequilibrium correction — measures how much the chain’s fluctuation structure exceeds that of its reversibilisation.

The result below shows that in reduced Fisher coordinates this correction factors exactly as a Gram operator built from the skew part of the generator. The factorisation follows from completing a quadratic form in the Donsker–Varadhan variational envelope.

Definitions

Let QQ be the generator of an irreducible continuous-time Markov chain on {1,…,n}\{1,\dots,n\} with stationary distribution π\pi, so π i>0\pi_i \gt 0 for all ii and πQ=0\pi Q = 0.

Fisher-conjugated generator. Set D π=diag(π 1,…,π n)D_\pi = \mathrm{diag}(\pi_1,\dots,\pi_n) and define

L π=D π 1/2QD π −1/2.L_\pi \;=\; D_\pi^{1/2}\, Q\, D_\pi^{-1/2}.

In components, (L π) ij=π iQ ij/π j(L_\pi)_{ij} = \sqrt{\pi_i}\, Q_{ij} / \sqrt{\pi_j}.

Euclideanised tangent space. Set e π=(π 1,…,π n) Te_\pi = (\sqrt{\pi_1},\dots,\sqrt{\pi_n})^T. The Euclideanised Fisher tangent space at π\pi is

T π={u∈ℝ n:⟨u,e π⟩=0}.T_\pi \;=\; \{u \in \mathbb{R}^n : \langle u, e_\pi \rangle = 0\}.

The orthogonal projector onto T πT_\pi is P π=I−e πe π TP_\pi = I - e_\pi e_\pi^T. The space T πT_\pi is L πL_\pi-invariant. After this change of variables the Fisher information metric at π\pi becomes the standard Euclidean inner product, so all orthogonality and adjointness below are Euclidean.

Reduced coordinates. Choose an orthonormal matrix C∈ℝ n×(n−1)C \in \mathbb{R}^{n \times (n-1)} whose columns span T πT_\pi, so C TC=I n−1C^T C = I_{n-1} and CC T=P πC C^T = P_\pi. Define the reduced Fisher-conjugated generator and its decomposition by

K^=C TL πC,G^=12(K^+K^ T),J^=12(K^−K^ T).\widehat{K} = C^T L_\pi\, C, \qquad \widehat{G} = \tfrac{1}{2}(\widehat{K} + \widehat{K}^T), \qquad \widehat{J} = \tfrac{1}{2}(\widehat{K} - \widehat{K}^T).

Here G^\widehat{G} is the reduced dissipative backbone (symmetric, negative definite) and J^\widehat{J} is the reduced skew channel (skew-symmetric). The chain satisfies detailed balance if and only if J^=0\widehat{J} = 0.

Donsker–Varadhan Hessian. The Donsker–Varadhan rate function? for the empirical measure is

I DV(μ)=sup h(−∑ i,jμ iQ ij(e h j−h i−1))I_{DV}(\mu) \;=\; \sup_{h}\Bigl(-\sum_{i,j} \mu_i\, Q_{ij}\,(e^{h_j - h_i} - 1)\Bigr)

with I DV(π)=0I_{DV}(\pi) = 0. Its Hessian at π\pi in reduced Fisher coordinates is the positive-definite (n−1)×(n−1)(n{-}1) \times (n{-}1) matrix H DVH_{DV}.

Detailed-balance Hessian. The canonical detailed-balance reference is the chain with Fisher-conjugated generator equal to the symmetric part G^\widehat{G}. Its reduced Donsker–Varadhan Hessian is

H 0=−12G^.H_0 \;=\; -\tfrac{1}{2}\,\widehat{G}.

This is positive definite (since G^\widehat{G} is negative definite on the reduced space).

Nonequilibrium correction.

Δ DV=H DV−H 0.\Delta_{DV} \;=\; H_{DV} - H_0.

Statement

Theorem (Dunkley 2025)

In reduced Euclidean Fisher coordinates at π\pi,

Δ DV=14J^H 0 −1J^ T.\Delta_{DV} \;=\; \tfrac{1}{4}\;\widehat{J}\;H_0^{-1}\;\widehat{J}^{\,T}.
Proof

Set S=−G^S = -\widehat{G}, so SS is positive definite and H 0=12SH_0 = \frac{1}{2}S. The reduced Donsker–Varadhan envelope expanded to second order in reduced Fisher coordinates xx and reduced logarithmic coordinates yy gives

ℒ(x,y)=−y TSy+x T(S+J^)y+O(3)\mathcal{L}(x,y) = -y^T S\, y + x^T(S + \widehat{J})\,y + O(3)

using K^=G^+J^=−S+J^\widehat{K} = \widehat{G} + \widehat{J} = -S + \widehat{J} and the fact that y TJ^y=0y^T \widehat{J}\,y = 0 for skew-symmetric J^\widehat{J}. Completing the square in yy:

ℒ(x,y)=−(y−12S −1(S+J^)x) TS(y−12S −1(S+J^)x)+14x T(S+J^) TS −1(S+J^)x+O(3).\mathcal{L}(x,y) = -\Bigl(y - \tfrac{1}{2} S^{-1}(S + \widehat{J})x\Bigr)^T S \Bigl(y - \tfrac{1}{2} S^{-1}(S + \widehat{J})x\Bigr) + \tfrac{1}{4}\, x^T (S + \widehat{J})^T S^{-1} (S + \widehat{J})\, x + O(3).

Supremising over yy:

I DV(μ(x))=14x T(S+J^) TS −1(S+J^)x+O(|x| 3).I_{DV}(\mu(x)) = \tfrac{1}{4}\, x^T (S + \widehat{J})^T S^{-1}(S + \widehat{J})\, x + O(|x|^3).

Since J^ T=−J^\widehat{J}^T = -\widehat{J}:

(S+J^) TS −1(S+J^)=(S−J^)S −1(S+J^)=S−J^S −1J^=S+J^S −1J^ T.(S + \widehat{J})^T S^{-1}(S + \widehat{J}) = (S - \widehat{J})S^{-1}(S + \widehat{J}) = S - \widehat{J}\,S^{-1}\widehat{J} = S + \widehat{J}\,S^{-1}\widehat{J}^T.

Therefore I DV(μ(x))=14x T(S+J^S −1J^ T)x+O(|x| 3)I_{DV}(\mu(x)) = \frac{1}{4} x^T(S + \widehat{J}\,S^{-1}\widehat{J}^T)\,x + O(|x|^3). Since I DV(μ(x))=12x TH DVx+O(|x| 3)I_{DV}(\mu(x)) = \frac{1}{2} x^T H_{DV}\, x + O(|x|^3), we get

H DV=12S+12J^S −1J^ T.H_{DV} = \tfrac{1}{2} S + \tfrac{1}{2}\widehat{J}\,S^{-1}\widehat{J}^T.

Now H 0=12SH_0 = \frac{1}{2}S and H 0 −1=2S −1H_0^{-1} = 2S^{-1}, giving

Δ DV=H DV−H 0=12J^S −1J^ T=14J^H 0 −1J^ T.\Delta_{DV} = H_{DV} - H_0 = \tfrac{1}{2}\widehat{J}\,S^{-1}\widehat{J}^T = \tfrac{1}{4}\widehat{J}\,H_0^{-1}\widehat{J}^T.

Signal operator form

Writing A DV=12J^H 0 −1/2A_{DV} = \frac{1}{2}\,\widehat{J}\,H_0^{-1/2}, the correction takes Gram matrix? form:

Δ DV=A DVA DV T.\Delta_{DV} = A_{DV}\,A_{DV}^T.

Consequences

Positivity and rank. Since Δ DV=A DVA DV T\Delta_{DV} = A_{DV} A_{DV}^T, positivity is manifest. Because H 0H_0 is invertible, rank(Δ DV)=rank(J^)\mathrm{rank}(\Delta_{DV}) = \mathrm{rank}(\widehat{J}).

Detailed-balance characterisation. The following are equivalent: (1) the chain satisfies detailed balance; (2) J^=0\widehat{J} = 0; (3) Δ DV=0\Delta_{DV} = 0.

Optimal rank-dd observation. The maximum trace of Δ DV\Delta_{DV} retained by any rank-dd orthogonal projection is 14∑ k=1 dσ k 2\frac{1}{4}\sum_{k=1}^{d} \sigma_k^2, where σ 1≥⋯≥σ r>0\sigma_1 \geq \cdots \geq \sigma_r \gt 0 are the nonzero singular values of A DVA_{DV}. This is a direct application of the Eckart–Young–Mirsky theorem?.

Vanishing criterion. The nonequilibrium correction is invisible to a rank-dd observer if and only if the observer’s subspace is orthogonal to the top dd left singular vectors of A DVA_{DV}.
