Definition

Definition

In a category, an object is called a zero object, or null object if it is both an initial object and a terminal object.

A category with a zero object is sometimes called a pointed category.

Remark

This means that 0‚ààùíû0 \in \mathcal{C} is a zero object precisely if for every other object AA there is a unique morphism A‚Üí0A \to 0 to the zero object as well as a unique morphism 0‚ÜíA0 \to A from the zero object.

Remark

There is also a notion of zero object in algebra which does not always coincide with the category-theoretic terminology. For example the zero ring {0}\{0\} is not an initial object in the category of unital rings (this is instead the integers ‚Ñ§\mathbb{Z}); but it is the terminal object. However, the zero ring is the zero object in the category of nonunital rings (although it happens to be unital).

Examples

Proposition

  • The category of pointed sets has a zero object, namely any one-element set.

  • The trivial group is a zero object in the category Grp of groups and in the category Ab of abelian groups.

  • For RR a ring, the trivial RR-module (that whose underlying abelian group is the trivial group) is the zero-object in RRMod.

    In particular for R=kR = k a field, the kk-vector space of dimension 0 is the zero object in Vect.

  • However, the zero ring is not a zero object in the category of rings, at least as long as rings are required to have units (and ring homomorphisms to preserve them).

  • For every category CC with a terminal object ** the under category pt‚ÜìCpt \downarrow C of pointed objects in CC has a zero object: the morphism Id ptId_{pt}.

Proposition

In any category CC enriched over the category of pointed sets (Set *,‚àß)(Set_*, \wedge) with tensor product the smash product, any object that is either initial or terminal is automatically both and hence a zero object.

Then C(x,t)=*C(x,t) = * for all xx and so in particular C(t,t)=*C(t,t) = * and hence the identity morphism on tt is the basepoint in the pointed hom-set.
