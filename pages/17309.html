Idea

The extra structure of a model category over a category with weak equivalences induces concrete constructions for expressing homotopy between morphisms. These lead in particular to an explicit construction of the homotopy category of a model category.

Definition

Definition

Let ùíû\mathcal{C} be a model category and X‚ààùíûX \in \mathcal{C} an object.

  • A path object Path(X)Path(X) for XX is a factorization of the diagonal ‚àá X:X‚ÜíX√óX\nabla_X \colon X \to X \times X as
‚àá X:X‚ü∂‚ààWiPath(X)‚ü∂(p 0,p 1)X√óX. \nabla_X \;\colon\; X \underoverset{\in W}{i}{\longrightarrow} Path(X) \overset{(p_0,p_1)}{\longrightarrow} X \times X \,.

where X‚ÜíPath(X)X\to Path(X) is a weak equivalence. This is called a good path object if in addition Path(X)‚ÜíX√óXPath(X) \to X \times X is a fibration.

  • A cylinder object Cyl(X)Cyl(X) for XX is a factorization of the codiagonal (or ‚Äúfold map‚Äù) Œî XX‚äîX‚ÜíX\Delta_X X \sqcup X \to X as
Œî X:X‚äîX‚ü∂(i 0,i 1)Cyl(X)‚ü∂p‚ààWX. \Delta_X \;\colon\; X \sqcup X \overset{(i_0,i_1)}{\longrightarrow} Cyl(X) \underoverset{p}{\in W}{\longrightarrow} X \,.

where Cyl(X)‚ÜíXCyl(X) \to X is a weak equivalence. This is called a good cylinder object if in addition X‚äîX‚ÜíCyl(X)X \sqcup X \to Cyl(X) is a cofibration.

Remark

By the factorization axioms every object in a model category has both a good path object and as well as a good cylinder object according to def. 1. But in some situations one is genuinely interested in using non-good such objects.

For instance in the classical model structure on topological spaces, the obvious object X√ó[0,1]X\times [0,1] is a cylinder object, but not a good cylinder unless XX itself is cofibrant (a cell complex in this case).

More generally, the path object Path(X)Path(X) of def. 1 is analogous to the powering ‚ãî(I,X)\pitchfork(I,X) with an interval object and the cyclinder object Cyl(X)Cyl(X) is analogous to the tensoring with a cylinder object I‚äôXI\odot X. In fact, if ùíû\mathcal{C} is a VV-enriched model category and XX is fibrant/cofibrant, then these powers and copowers are in fact examples of (good) path and cylinder objects if the interval object is sufficiently good.

Lemma

  • Let XX be cofibrant. If there is a left homotopy f‚áí Lgf \Rightarrow_L g then there is also a right homotopy f‚áí Rgf \Rightarrow_R g (def. 2) with respect to any chosen path object.

Equivalence relation

Proposition

For XX a cofibrant object in a model category and YY a fibrant object, then the relations of left homotopy f‚áí Lgf \Rightarrow_L g and of right homotopy f‚áí Rgf \Rightarrow_R g (def. 2) on the hom set Hom(X,Y)Hom(X,Y) coincide and are both equivalence relations.

Proof

That both relations coincide under the (co-)fibrancy assumption follows directly from lemma 3.

To see that left homotopy with domain XX is a transitive relation first use lemma 2 to obtain that every left homotopy is exhibited by a good cylinder object Cyl(X)Cyl(X) and then lemma 1 to see that the cofiber coproduct Cyl(X)‚äîXCyl(X)Cyl(X)\underset{X}{\sqcup} Cyl(X) in
