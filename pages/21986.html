Idea

Recall that a topos is a category that behaves likes the category Set of sets.

An integers object internal to a topos is an object that behaves in that topos like the set ℤ\mathbb{Z} of integers does in Set.

Definition

In a topos or cartesian closed category

An integers object in a topos (or any cartesian closed category) EE with terminal object 11 is

  • an object ℤ\mathbb{Z} in EE

  • equipped with

    • a morphism z:1→ℤz:1 \to \mathbb{Z} from the terminal object 11;

    • morphisms s:ℤ→ℤs : \mathbb{Z} \to \mathbb{Z} (successor) and p:ℤ→ℤp : \mathbb{Z} \to \mathbb{Z} (predecessor), such that the following diagram commutes:

  • such that for every other object AA with morphisms z A:1→Az_A : 1 \to A and s A,p A:A→As_A, p_A : A \to A that satisfies the following commutative diagram:

there is a unique morphism u:ℤ→Au : \mathbb{Z} \to A such that u∘z=z Au \circ z = z_A, u∘s=s A∘uu \circ s = s_A \circ u, and u∘p=p A∘uu \circ p = p_A \circ u.

By the universal property, the integers object is unique up to isomorphism.

Properties

Negation

There is a nontrivial isomorphism on ℤ\mathbb{Z}, neg:ℤ→ℤneg:\mathbb{Z}\rightarrow\mathbb{Z}, such that neg∘z=zneg \circ z = z, s∘neg=neg∘ps \circ neg = neg \circ p, and p∘neg=neg∘sp \circ neg = neg \circ s. This corresponds to negation of integers in Set.

Relation to ring objects

The initial ring object in a category with finite products, an object ℤ\mathbb{Z} with global elements 0:1→ℤ0:1\rightarrow\mathbb{Z} and 1:1→ℤ1:1\rightarrow\mathbb{Z}, a morphism −:ℤ→ℤ-:\mathbb{Z}\rightarrow\mathbb{Z}, morphsims +:ℤ×ℤ→ℤ+:\mathbb{Z}\times\mathbb{Z}\rightarrow\mathbb{Z} and ×:ℤ×ℤ→ℤ)\times:\mathbb{Z}\times\mathbb{Z}\rightarrow\mathbb{Z}), and suitable commutative diagrams expressing the ring axioms and initiality, has the structure of an integers object given by the tuple (ℤ,0:1→ℤ,x+1:ℤ→ℤ,x+(−1):ℤ→ℤ)(\mathbb{Z}, 0:1\rightarrow\mathbb{Z}, x+1:\mathbb{Z}\rightarrow\mathbb{Z}, x+(-1):\mathbb{Z}\rightarrow\mathbb{Z}).
