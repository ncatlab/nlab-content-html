Definition

A global field is either

  • a number field

  • or a function field over a finite field;

hence is a finite-dimensional field extension of either

  • the rational functions ùîΩ q(t)\mathbb{F}_q(t) with coefficients in the finite field ùîΩ q\mathbb{F}_q;

  • the rational numbers ‚Ñö\mathbb{Q}.

The function field analogy says how both these kinds of global fields are indeed analogous in that they behave have as rings of rational functions on arithmetic curves over ùîΩ q\mathbb{F}_q and ‚Äúover F1‚Äù, respectively. In particular for both types there is a concept of ring of adeles, group of ideles, etc.

The Langlands conjectures concern the arithmetic geometry of global fields.

The ‚Äúlocal‚Äù version of a global field in the sense of formal geometry is a local field. The corresponding version of the Langlands program are the local Langlands correspondences.

Global fields can also be described axiomatically, as those fields which satisfy a suitable product formula. For example, if kk is a number field and ùïÄ k\mathbb{I}_k is its group of ideles, then there is an idele norm map

For the following theorem, assume that kk is a field for which there is at least one valuation that is either archimedean, or is discrete and with finite residue class field for the corresponding local ring. By a place we understand an equivalence class of valuations.

Theorem

(Artin-Whaples) Suppose kk is a field that has a set of valuations |(‚àí)| v{|(-)|}_v, one for every place of kk, such that for every x‚ààk √óx \in k^\times the product formula ‚àè v|x| v=1\prod_v {|x|}_v = 1 is satisfied. Then kk is a global field (and conversely, every global field has this property).
