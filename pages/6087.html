Definition

A Lie groupoid (C 1→t→sC 0)(C_1 \stackrel{\overset{s}{\to}}{\underset{t}{\to}} C_0) is proper if its underlying topological groupoid is a proper topological groupoid, hence if

(s,t):C 1→C 0×C 0 (s,t) : C_1 \to C_0 \times C_0

is a proper map.

So in particular the automorphism group of any object in a proper Lie groupoid is a compact Lie group. In this sense proper Lie groupoids generalize compact Lie groups.

Examples

  • An orbifold is (presented by) a proper étale groupoid.
