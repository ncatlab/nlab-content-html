The map v i:A→E iv_i:A\to E_i belongs to ℳ \mathcal{M}, since u i∈ℳu_i\in \mathcal{M} by assumption, and since the class ℳ \mathcal{M} is closed under cobase change. Hence the object (E i,v i)(E_i,v_i) of the category A\EA\backslash \mathbf{E} is cofibrant with respect to the class ℳ′ \mathcal{M}'. It follows that the object (B,u)(B,u) is cofibrant by the first part of the proof. This proves that u∈ℳu\in \mathcal{M}.

Definition

If α\alpha is a regular cardinal, we shall say that a class of maps ℳ\mathcal{M} in a complete category E\mathbf{E} is α\alpha-cellular if it satisfies the following conditions: * ℳ\mathcal{M} contains the isomorphisms and is closed under composition; * ℳ\mathcal{M} is closed under transfinite compositions of cocontinuous chains of length ≤α\le \alpha; * ℳ\mathcal{M} is closed under cobase changes; * ℳ\mathcal{M} is closed under coproducts.

We shall say that an α\alpha-cellular class 𝒞\mathcal{C} is α\alpha-saturated if in addition,

  • 𝒞\mathcal{C} is closed under retracts.

Every class of maps Σ⊆E\Sigma\subseteq \mathbf{E} is contained in a smallest α\alpha-cellular class Cell α(Σ)Cell^\alpha(\Sigma) called the α\alpha-cellular class generated by Σ\Sigma. Similarly, Σ\Sigma is contained in a smallest α\alpha-saturated class Sat(Σ)Sat(\Sigma) called the α\alpha-saturated class generated by Σ\Sigma.

Example

If Σ\Sigma is the set of inclusions ∂Δ[n]⊂Δ[n]\partial \Delta[n] \subset \Delta[n] (n≥0n\geq 0) in the category SSet\mathbf{SSet} (of simplicial sets), then Cell ω(Σ)=Sat(Σ)Cell^\omega(\Sigma)=Sat(\Sigma) is the class of monomorphisms.

Example

If Σ\Sigma is the set of inclusions h n k:Λ k[n]⊂Δ[n]h^k_n: \Lambda^k[n] \subset \Delta[n] for n>0n\gt 0 and 0≤k≤n0\le k\le n, then Sat ω(Σ)=Sat(Σ)Sat^\omega(\Sigma)=Sat(\Sigma) is the class of anodyne maps.
