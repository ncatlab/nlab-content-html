Examples

  • The category Prop of propositions is enriched over itself. Hence for any two proposition a,ba,b, there is a hom-proposition Prop(A,B)Prop(A,B). This is the implication proposition a‚áíba \implies b.

  • The category Set of sets is enriched over itself. Hence for any two sets A,BA,B, there is a hom-set Set(A,B)Set(A,B). This is the function set A‚ÜíBA \to B.

  • The category Grpd of groupoids is enriched over itself. Hence for any two groupoids A,BA,B, there is a hom-groupoid Grpd(A,B)Grpd(A,B). This is the functor category Func(A,B)Func(A,B).

  • The category Pos of posets is enriched over itself. Hence for any two posets A,BA,B, there is a hom-poset Pos(A,B)Pos(A,B), the poset of monotones.
