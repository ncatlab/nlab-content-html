Definitions and framework

A bineary linear code is a linear code over the prime field 𝔽 2\mathbb{F}_2.

Linear codes

A (qq-ary) linear code of length nn and rank kk is a kk-dimensional subspace of the nn-dimensional vector space 𝔽 q n\mathbb{F}_q^n over a prime field 𝔽 q\mathbb{F}_q with qq elements, or of any nn-dimensional space over 𝔽 q\mathbb{F}_q that comes equipped with a “standard” basis. A principal application is in the construction of error-correcting codes.

There is a Joyal species Code q,k:Core(FinSet)→SetCode_{q, k}: Core(FinSet) \to Set of qq-ary linear codes of rank kk, which assigns to a finite set Ω\Omega the set of kk-dimensional subspaces of the free 𝔽 q\mathbb{F}_q-vector space generated by Ω\Omega as basis. The corresponding groupoid of qq-linear codes is the category of elements of Code q,kCode_{q, k}. This gives one notion of isomorphism of linear codes, where an isomorphism (S,V⊆𝔽 qS)→(T,W⊆𝔽 qT)(S, V \subseteq \mathbb{F}_q S) \to (T, W \subseteq \mathbb{F}_q T) is a bijection ϕ:S→T\phi: S \to T such that the linear map 𝔽 qϕ\mathbb{F}_q \phi carries VV onto WW. Thus, isomorphisms in this sense are given by permutation matrices.

An alternative notion that recurs in coding theory is that an isomorphism (often called an equivalence) of codes consists of a monomial matrix which, when interpreted as a linear isomorphism using the assigned linear bases, carries VV onto WW.

Each assigned basis {e 1,…,e n}\{e_1, \ldots, e_n\} of a vector space UU can be construed as a self-dual basis with respect to a bilinear form, so that ⟨e i,e j⟩=δ ij\langle e_i, e_j \rangle = \delta_{i j}. In terms of coordinates with respect to the chosen basis, this bilinear form is the usual dot product

(x→,y→)↦x 1y 1+…+x ny n(\vec{x}, \vec{y}) \mapsto x_1 y_1 + \ldots + x_n y_n

and this bilinear form is used to define the notion of dual code (see below).

Binary linear codes

The following is largely adapted from Frenkel, Lepowsky, Meurman.

A (binary linear) code is a qq-ary code with q=2q = 2. In this case, isomorphisms defined by permutation matrices are the same as isomorphisms defined by monomial matrices.

As a combinatorial structure on a finite set Ω\Omega, a binary linear code is typically construed as a subspace of a finite power set P(Ω)P(\Omega), considered as an 𝔽 2\mathbb{F}_2-space whose addition is given by taking the symmetric difference of sets, and with standard basis given by the atoms.

In that case, the standard dot product can be identified with the nondegenerate symmetric 𝔽 2\mathbb{F}_2-bilinear pairing on the Boolean ring P(Ω)P(\Omega) given by

P(Ω)×P(Ω)→⟨−,−⟩𝔽 2 P(\Omega) \times P(\Omega) \stackrel{\langle -, - \rangle}{\to} \mathbb{F}_2
\,
(S,T)↦|S∩T|(mod2)(S, T) \mapsto {|S \cap T|} \pmod 2

where |C|{|C|} is the cardinality of a set CC. In the sequel, we let n=|Ω|=dim(P(Ω))n = {|\Omega|} = \dim(P(\Omega)).

To each code C⊂P(Ω)\mathbf{C} \subset P(\Omega) there is a dual code C ⊥\mathbf{C}^\perp:

C ⊥={S∈P(Ω):∀T∈C⟨S,T⟩=0}.\mathbf{C}^\perp = \{S \in P(\Omega): \forall T \in \mathbf{C} \; \langle S, T \rangle = 0\}.

Notice that dim(C ⊥)=n−dim(C)\dim(\mathbf{C}^\perp) = n - \dim(\mathbf{C}). A code C\mathbf{C} is self-dual if C=C ⊥\mathbf{C} = \mathbf{C}^\perp. In this case nn is even and dim(C)=n/2\dim(\mathbf{C}) = n/2.

Now suppose Ω\Omega is a set with an even number nn of elements. The collection E(Ω)E(\Omega) of subsets of even cardinality is a vector subspace of P(Ω)P(\Omega) of dimension n−1n-1, and we have

E(Ω)=1 ⊥,E(\Omega) = 1^\perp,

where 11 is the multiplicative unit of P(Ω)P(\Omega) (the subset Ω\Omega), under the assumption nn is even. On E(Ω)E(\Omega) we may define a quadratic form

q:E(Ω)→𝔽 2q: E(\Omega) \to \mathbb{F}_2

taking S∈E(Ω)S \in E(\Omega) to |S|2(mod2)\frac{{|S|}}{2} \pmod 2; this qq gives rise to the bilinear form ⟨−,−⟩\langle -, - \rangle on E(Ω)E(\Omega) via

⟨S,T⟩=q(S+T)−q(S)−q(T).\langle S, T \rangle = q(S + T) - q(S) - q(T).

A code C⊂P(Ω)\mathbf{C} \subset P(\Omega) is of type II if

  • 44 divides |C|{|C|} for every C∈CC \in \mathbf{C} and Ω∈C\Omega \in \mathbf{C},

and of type I if

  • C⊂E(Ω)\mathbf{C} \subset E(\Omega) and Ω∈C\Omega \in \mathbf{C}, but C\mathbf{C} is not of type II.

Notice that the quadratic form (hence the bilinear form) vanishes on a type II code C\mathbf{C}, so that we have C⊂C ⊥\mathbf{C} \subset \mathbf{C}^\perp (i.e., C\mathbf{C} is isotropic). If C\mathbf{C} is maximal isotropic – has half the dimension of P(Ω)P(\Omega) – then it follows that C\mathbf{C} is self-dual.

Examples

Example

(Hamming (8,4)(8, 4) code) Up to isomorphism, there is a unique type II self-dual code C⊂P(Ω)\mathbf{C} \subset P(\Omega) where Ω\Omega is an 8-element set. Such a code may be constructed as follows: let Ω\Omega be the projective line ℙ 1𝔽 7\mathbb{P}^1 \mathbb{F}_7, so as a set Ω={0,1,2,3,4,5,6,∞}\Omega = \{0, 1, 2, 3, 4, 5, 6, \infty\}. Let QQ be the set of squares 0,1,2,40, 1, 2, 4 and let NN be its complement in Ω\Omega. The span of the set of elements {N+i:i∈𝔽 7}\{N + i: i \in \mathbb{F}_7\} (where ∞+i=∞\infty + i = \infty for each ii) is a 4-dimensional space consisting of 16 elements:

∅, {3,5,6,∞}, {0,4,6,∞}, {0,1,5,∞} {1,2,6,∞}, {0,2,3,∞}, {1,3,4,∞}, {2,4,5,∞} {0,3,4,5}, {1,4,5,6}, {0,2,5,6}, {0,1,3,6} {0,1,2,4}, {1,2,3,5}, {2,3,4,6}, Ω.\array{ \emptyset, & \{3, 5, 6, \infty\}, & \{0, 4, 6, \infty\}, & \{0, 1, 5, \infty\} \\ \{1, 2, 6, \infty\}, & \{0, 2, 3, \infty\}, & \{1, 3, 4, \infty\}, & \{2, 4, 5, \infty\} \\ \{0, 3, 4, 5\}, & \{1, 4, 5, 6\}, & \{0, 2, 5, 6\}, & \{0, 1, 3, 6\} \\ \{0, 1, 2, 4\}, & \{1, 2, 3, 5\}, & \{2, 3, 4, 6\}, & \Omega. }

For this subspace C⊂P(Ω)\mathbf{C} \subset P(\Omega), every element is a set that is of cardinality 00, 44, or 88. It is clearly of type II, and self-dual because it is maximal isotropic.

There is a second Hamming code C′\mathbf{C}' (isomorphic to C\mathbf{C}) that is spanned by the elements {−N−i:i∈𝔽 7}\{-N - i: i \in \mathbb{F}_7\}. This is “complementary” to the first in an appropriate sense: we have

C+C′=E(Ω),C∩C′=𝔽 2Ω,\mathbf{C} + \mathbf{C}' = E(\Omega), \qquad \mathbf{C} \cap \mathbf{C}' = \mathbb{F}_2 \Omega,

so that C/𝔽 2Ω\mathbf{C}/\mathbb{F}_2 \Omega is complementary to C′/𝔽 2Ω\mathbf{C}'/\mathbb{F}_2 \Omega in the 6-dimensional space E(Ω)/𝔽 2ΩE(\Omega)/\mathbb{F}_2 \Omega.

In the theory of error-correcting codes, this is related to the Hamming (7,4)(7, 4) code, but with an extra parity bit. The Hamming (7,4)(7, 4) code is one of an infinite family of Hamming codes of type (2 r−1,2 r−r−1)(2^r - 1, 2^r - r - 1).

Example

(The binary Golay code.) Up to isomorphism, there is a unique self-dual code of type II on a 24-element set with no elements of weight (= cardinality) 4. Indeed, let 3Ω3 \Omega denote the disjoint union of three copies of Ω\Omega. Construct a code D⊂P(3Ω)\mathbf{D} \subset P(3\Omega) that is spanned by elements of the form

S⊔S⊔∅,S⊔∅⊔S,T⊔T⊔TS \sqcup S \sqcup \emptyset, \qquad S \sqcup \emptyset \sqcup S, \qquad T \sqcup T \sqcup T

where SS ranges over elements of the Hamming code C\mathbf{C} and TT ranges over elements of the Hamming code C′\mathbf{C}'. Any three elements S⊔S⊔∅S \sqcup S \sqcup \emptyset, S′⊔∅⊔S′S' \sqcup \emptyset \sqcup S', and T⊔T⊔TT \sqcup T \sqcup T are mutually orthogonal in P(Ω)P(\Omega), so evidently D\mathbf{D} is an orthogonal direct sum of three 4-dimensional isotropic spaces, making D\mathbf{D} an isotropic subspace of dimension 12, hence maximal isotropic. Each element of D\mathbf{D} has cardinality a multiple of 44, making it self-dual type II.

To see that D\mathbf{D} has no elements of weight 4, argue as follows. Note that combinations (S⊔S⊔∅)+(S′⊔∅⊔S′)(S \sqcup S \sqcup \emptyset) + (S' \sqcup \emptyset \sqcup S') are of the form S 1⊔S 2⊔S 3S_1 \sqcup S_2 \sqcup S_3 such that S 1+S 2+S 3=0S_1 + S_2 + S_3 = 0. Now suppose we had

|S 1+T|+|S 2+T|+|S 3+T|=4.{|S_1 + T|} + {|S_2 + T|} + {|S_3 + T|} = 4.

Since all the summands are even, one must be zero, so say S 3+T=0S_3 + T = 0. Since C∩C′=𝔽 2Ω\mathbf{C} \cap \mathbf{C}' = \mathbb{F}_2 \Omega, we must have T∈𝔽 2ΩT \in \mathbb{F}_2 \Omega. But then |S 1+T|{|S_1 + T|} and |S 2+T|{|S_2 + T|} are multiples of 44, and therefore one is zero, so say S 2+T=0S_2 + T = 0. But now we have

S 1=S 1−2T=S 1+S 2+S 3=0S_1 = S_1 - 2 T = S_1 + S_2 + S_3 = 0

whence we infer that |S 1+T|{|S_1 + T|} is a multiple of 88, and we have reached a contradiction.

Properties

Relation to quantum stabilizer codes

Classical binary linear codes are closely related to quantum error correcting codes knows as stabilizer codes (e.g. Ball, Centelles & Huber 20, Sec. 2.3 & 3)
