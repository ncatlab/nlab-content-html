Definition

A strict nn-category is a strict omega-category all whose k-morphisms for k>nk \gt n are identities.

The category nCatn Cat of strict nn-categories and n-functors between them can also be defined inductively by

  • starting by setting 0Cat:=0 Cat := Set;

  • noticing that Set is canonically a (symmetric, in fact cartesian) closed monoidal category such that one can consider categories enriched over it;

  • noticing that for VV any complete and cocomplete closed monoidal category, also VCatV Cat (the category of VV-enriched categories) has these same properties;

  • finally setting, recursively,

    (n+1)Cat:=(nCat)Cat. (n+1)Cat := (n Cat) Cat \,.

The category StrœâCatStr\omega Cat of strict œâ\omega-categories can then in turn be defined as a suitable colimit of the categories nCatn Cat.

Examples

A strict 1-category is just a category.

Strict 2-categories are important, because the coherence theorem for bicategories states that every (‚Äúweak‚Äù) 2-category is equivalent to a strict one, and also because many 2-categories, such as Cat, are naturally strict. However, for n‚â•3n\ge 3, these two properties fail, so that strict nn-categories become less useful (though not useless). Instead, one needs to use (at least) semistrict categories.
