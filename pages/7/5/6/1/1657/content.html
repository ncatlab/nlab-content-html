Idea

In measure theory, measure spaces are used in the general theory of measure and integration, somewhat analogous to the role played by topological spaces in the study of continuity.

For the general theory of measure spaces, we first need a measurable space (X,Σ)(X, \Sigma), that is a set equipped with a collection Σ\Sigma of measurable sets complete under certain operations. Then this becomes a measure space (X,Σ,μ)(X, \Sigma, \mu) by throwing in a function μ\mu from Σ\Sigma to a space of values (such as the real line) that gets along with the set-theoretic operations that Σ\Sigma has. If EE is a measurable set, then μ(E)\mu(E) is called the measure of EE with respect to μ\mu.

Notation

The original notation for an integral (going back to Gottfried Leibniz) was

(1)∫ a bf(x)dx \int_a^b f(x) \,\mathrm{d}x

(where f(x)f(x) would be replaced by some formula in the variable xx). In modern measure theory, we can now understand this as the integral of the measurable function ff on the interval [a,b][a,b] relative to Lebesgue measure. If we wish to generalise from Lebesgue measure to an arbitrary measure μ\mu and generalise from [a,b][a,b] to an arbitrary measurable set SS, then we can write

(2)∫ x∈Sf(x)μ(dx) \int_{x\in{S}} f(x) \,\mu(\mathrm{d}x)

instead. Now, if ff is not given by a formula but rather explicitly named, then there is no need for the dummy variable xx, so we should write

(3)∫ Sfμ. \int_S f \,\mu .

However, it has been more common to keep the symbol ‘d\mathrm{d}’ and write

(4)∫ Sfdμ. \int_S f \,\mathrm{d}\mu .

(Note that ‘d\mathrm{d}’ can be read as ‘with respect to’ in both (1) and (4), although meaning different things; in the former case, it indicates the dummy variable, while in the latter case, it indicates the measure.) This notation then leads to replacing (2) with

(5)∫ x∈Sf(x)dμ(x). \int_{x\in{S}} f(x) \,\mathrm{d}\mu(x) .

This last notation, however, hides the fact that integrating a function with respect to a measure is a way of multiplying a function by a measure to get a new measure; the integral of ff on SS with respect to μ\mu is simply the measure of SS with respect to fμf \mu, as can be seen in (3). Compare also notation for Radon-Nikodym derivatives.

It is also possible to take the entire expression ‘dμ\mathrm{d}\mu’ as the name of the measure, writing dμ(A)\mathrm{d}\mu(A) even where the common notation is μ(A)\mu(A). In that case, the common expression (4) is literally the same as (what would otherwise be) (3), although (5) is not quite the same as (what would otherwise be) (2).

We will use (3) below (although other forms may well be found on other pages).

See Usenet discussion, and contrast (5) with the Stieltjes integral. (The point is that it is dx\mathrm{d}x, not just xx, that gives us the relevant measure in (1).) The notation (3) has also been used in an introductory graduate-level course by John Baez.

There is also some variation in notation as to whether to use a roman ‘d\mathrm{d}’ or an italic ‘d\mathit{d}’; roman is more common in England and italic in America. But of course, that variation should not cause any difficulties!

Definitions

A measure space is a measurable space equipped with a measure. There are many different kinds of measures; we start with the most specific and then consider generalisations. The motivating example is Lebesgue measure on the unit interval.

Probability measures

Let (X,Σ)(X, \Sigma) be a measurable space. A probability measure on XX (due to Andrey Kolmogorov) is a function μ\mu from the collection Σ\Sigma of measurable sets to the unit interval [0,1][0,1] such that:

  1. The measure of the entire space is one: μ(X)=1\mu(X) = 1;
  2. Countable additivity: μ(⋃ i=1 ∞S i)=∑ i=1 ∞μ(S i)\mu(\bigcup_{i = 1}^{\infty} S_i) = \sum_{i=1}^{\infty} \mu(S_i) whenever the S iS_i are mutually disjoint.

(Part of the latter condition is the requirement that the sum on the right-hand side must converge.)

It is sometimes stated (but in fact follows from the above) that:

  • Finitary additivity: μ(S∪T)=μ(S)+μ(T)\mu(S \cup T) = \mu(S) + \mu(T) whenever SS and TT are disjoint.
  • μ\mu is increasing: μ(A)≤μ(B)\mu(A) \leq \mu(B) if A⊆BA \subseteq B.
  • The measure of the empty set is zero: μ(∅)=0\mu(\emptyset) = 0;

The first of these conditions will follow for all of the generalised notions of measure below, but the second usually will not. Related query discussion is archived here.

Generalizations

From now on, we drop the condition μ(X)=1\mu(X)=1; the next step is to generalize the target of μ\mu, as follows:

  • Use [0,∞)[0,\infty) (instead of [0,1][0,1]) for a (finite) positive measure.
  • Use R=]−∞,∞[\mathbf{R} = {]{-\infty,\infty}[} for a (finite) signed measure (alias charge).
  • Use C\mathbf{C} for a (finite) complex-valued measure.
  • Use an arbitrary topological vector space VV for a vector-valued measure.
  • In principle, one could go further yet; VV just needs an analogue of addition with a notion of infinitary sum (such as a topological abelian group has). But until someone suggests a useful example, we will leave this to the centipedes.

We define an infinite measure by replacing the domain of μ\mu by an ideal Σ′\Sigma' of Σ\Sigma such that the following saturation condition is satisfied: if {S i} i∈I\{S_i\}_{i\in I} is a disjoint family of elements of Σ′\Sigma' and ∑ i∈I|μ|(S i)\sum_{i\in I}|\mu|(S_i) exists (and is finite), then ⋃ i∈IS i∈Σ′\bigcup_{i\in I}S_i\in\Sigma'. The countable additivity condition should now be modified to require ⋃ i∈IS i∈Σ′\bigcup_{i\in I}S_i\in\Sigma'.

An infinite measure μ\mu is semifinite if for any S∈Σ∖Σ′S\in\Sigma\setminus\Sigma' there is T∈Σ′T\in\Sigma' such that T⊂ST\subset S and μ(T)>0\mu(T)\gt0. The Radon-Nikodym theorem shows that semifinite complex-valued measures that are absolutely continuous with respect to some fixed localizable measure form a free module over the algebra of complex-valued measurable functions (not necessarily bounded).

Some further terms:

  • We can combine conditions; for example a finite positive measure takes values in [0,∞[{[{0,\infty}[}.
  • A measure is bounded if, for some (finite) real number MM, |μ(S)|≤M|\mu(S)| \leq M for every measurable set SS. (This requires that the target of μ\mu have a real-valued notion of absolute value or norm, so a vector-valued measure should be valued in a Banach space or something similar.)
  • A measure is σ\sigma-finite if every measurable set is a union of countably many sets with finite measure.

Remarks:

  • The property that μ\mu is increasing holds for all positive measures but may fail for others.
  • A positive measure μ\mu that satisfies μ(X)=1\mu(X)=1 must be a probability measure as defined earlier; that is, it satisfies μ(S)≤1\mu(S) \leq 1 for all SS.
  • When ∞\infty is allowed as a value of μ\mu, then the requirement in (3) that the sum converges should be interpreted in this light; that is, the sum may diverge to infinity. (For a positive measure, therefore, the convergence criterion is vacuous in classical mathematics.)
  • Notice that −∞-\infty is not allowed as a value for a signed measure. It would work just as well to allow −∞-\infty and forbid ∞\infty. It is even possible to allow both, but this is a little trickier (because of −∞+∞-\infty + \infty), so we deal with it later (at the end of this subsection).

Another possibility is to generalize the source of μ\mu; instead of using a σ\sigma-algebra on XX, we could use a σ\sigma-ring or even a δ\delta-ring. These versions are mostly more about changing the definition of measurable space, so refer there for details of the definitions; however, we note that (3), when Σ\Sigma is a δ\delta-ring, should state that the left-hand side exists (that is, the union is measurable) if the right-hand side converges. Generalizing Σ\Sigma in this way is complementary to generalizing the target above; in particular it may allow one to avoid dealing with ∞\infty. For example, while Lebesgue measure is only a positive measure on a σ\sigma-algebra, it is a finite positive measure on the δ\delta-ring of bounded measurable sets. Indeed, every signed measure gives rise to finite measure on its δ\delta-ring of finitely measurable sets (as defined below); conversely, every σ\sigma-finite measure can be recovered from this by imposing (3) in all cases.

Yet another possibility is to drop countable additivity, replacing it with finite additivity. The result is a finitely additive measure, sometimes called a charge to avoid the red herring principle; in contrast, the usual sort of measure may be called countably additive. For a charge, one could replace Σ\Sigma with an algebra (or even a ring) of sets; again see measurable space for these definitions.

Finally, an extended measure takes values in the set [−∞,∞][-\infty,\infty] of extended real numbers. Here we have the problem that, even when considering finite additivity, we might have to add ∞\infty and −∞-\infty. While we might simply require that this never happens (so that at least one of μ(S)\mu(S) and μ(T)\mu(T) must be finite if they have opposite signs and S∩T=∅S \cap T = \empty), this does not include some examples that we want (and in fact it would follow that ∞\infty and −∞-\infty cannot both be values of μ\mu after all). To deal with this, we define an extended measure to be a formal difference μ +−μ −\mu^+ - \mu^- of positive measures; μ(S)=μ +(S)−μ −(S)\mu(S) = \mu^+(S) - \mu^-(S) whenever this is not of the form ∞−∞\infty - \infty and is otherwise undefined. Note that the set of extended measures on XX is a quotient set of the set of pairs of positive measures; we say that μ=ν\mu = \nu if μ(S)=ν(S)\mu(S) = \nu(S) whenever either side is defined, that is if μ\mu and ν\nu are the same as partial functions from Σ\Sigma to [−∞,∞][-\infty,\infty].

Any extended measure restricts to an infinite signed measure, taking Σ′⊂Σ\Sigma'\subset\Sigma to be the set of elements of Σ\Sigma with a finite measure. Vice versa, an infinite signed measure μ\mu canonically extends to an extended measure: we can define μ +\mu_+ and μ −\mu_- as usual and then take the formal difference μ +−μ −\mu_+-\mu_-.

Point-free measure spaces

Just like how in point-free topology, one takes the opens to be fundamental and defines a locale as a frame of opens, in point-free measure theory, one takes the measurables to be fundamental and defines a measurable space as a σ \sigma -complete Boolean algebra of measurables.

Then the various notions of measures and measure spaces above also make sense on the σ\sigma-complete Boolean algebra, but where the countable union and intersection of measurable subsets in a traditional measurable space is replaced with the countable join and meet of measurables in the σ\sigma-complete Boolean algebra. For example, a probability measure on a σ\sigma-complete Boolean algebra Σ\Sigma is a function μ:Σ→[0,1]\mu:\Sigma \to [0, 1] such that

  1. The measure of the top element is one: μ(⊤)=1\mu(\top) = 1

  2. Countable additivity: μ(⋁ i∈ℕS i)=∑ i=0 ∞μ(S i)\mu(\bigvee_{i \in \mathbb{N}} S_i) = \sum_{i=0}^{\infty} \mu(S_i) whenever S i∧S j=⊥S_i \wedge S_j = \bot for all natural numbers i∈ℕi \in \mathbb{N} and j∈ℕj \in \mathbb{N} where i≠ji \neq j.

Constructive theory

In Henry Cheng's constructive theory of measure, the definition of measurable space becomes more complicated; the main point is that a single measurable set SS is replaced by a complemented pair (S,T)(S,T). Once that is understood, very little needs to be changed to define a measure space.

In the requirements (1–3), the constants ∅\empty and XX and the operation ∪\union are interpreted by formal de Morgan duality, as explained at Cheng measurable space. The convergence requirement in (3) should be interpreted in the strong sense of located convergence and is no longer trivial for positive measures. We must add a further requirement to enforce the idea that μ(S,T)\mu(S,T) is the measure of SS alone, as follows:

  • μ(S,T)=μ(S,U)\mu(S,T) = \mu(S,U) whenever (S,T)(S,T) and (S,U)(S,U) are both complemented pairs.

In general, a measurable set is any set SS such that (S,T)(S,T) is a complemented pair for some set TT; the term ‘measurable set’ in the classical theory should be interpreted as either ‘mesurable set’ or ‘complemented pair’ in the constructive theory, depending on context. Usually both interpretations will actually work, but often only the first set of the pair will matter, thanks to the axiom above.

We will mention other occasional fine points in the constructive theory when they occur; the main outline does not change.

Measures on a σ\sigma-frame

In measure theory, a measure on a σ \sigma -frame or more generally a σ \sigma -complete distributive lattice (L,≤,⊥,∨,⊤,∧,⋁)(L, \leq, \bot, \vee, \top, \wedge, \Vee) is a valuation μ:L→[0,∞]\mu:L \to [0, \infty] such that the elements are mutually disjoint and the probability valuation is denumerably/countably additive

∀s∈L ℕ.∀m∈ℕ.∀n∈ℕ.(m≠n)∧(s(m)∧s(n)=⊥)\forall s\in L^\mathbb{N}. \forall m \in \mathbb{N}. \forall n \in \mathbb{N}. (m \neq n) \wedge (s(m) \wedge s(n) = \bot)
∀s∈L ℕ.μ(⋁ n:ℕs(n))=∑ n:ℕμ(s(n))\forall s\in L^\mathbb{N}. \mu(\Vee_{n:\mathbb{N}} s(n)) = \sum_{n:\mathbb{N}} \mu(s(n))

Subsidiary definitions

Given a measure space (X,Σ,μ)(X,\Sigma,\mu), a μ\mu-null Σ\Sigma-measurable set is a measurable set NN such that μ(S)=0\mu(S) = 0 whenever S⊆NS \subseteq N is measurable; a μ\mu-null set is any subset of a null measurable set. In a positive measure space, we don't have to bother with SS; NN will be a null measurable set as long as μ(N)=0\mu(N) = 0.

A μ\mu-full Σ\Sigma-measurable set is a measurable set FF such that μ(S)=μ(S∩F)\mu(S) = \mu(S \cap F) for every measurable set SS; a μ\mu-full set is any superset of a full measurable set. In a probability measure space, we don't have to bother with SS; FF will be a full measurable set as long as μ(F)=1\mu(F) = 1. Classically, a full set is precisely the complement of a null set, but this doesn't hold in the constructive theory.

A property of elements of XX holds μ\mu-almost everywhere if the set of values where it holds is full.

A measure is complete if every full set is measurable. We may form the completion of a measure space by accepting as a measurable set the intersection of any set and a full set; these μ\mu-measurable sets will automatically form a σ\sigma-algebra (or whatever Σ\Sigma originally was). Classically, a measure is complete if and only if every null set is measurable and a set is μ\mu-measurable if and only if it is the symmetric difference between a measurable set and a null set.

A finitely μ\mu-measurable set is a measurable set MM such that μ(S)\mu(S) is finite whenever S⊆MS \subseteq M is measurable; a σ\sigma-finitely μ\mu-measurable set is any union of countably many finitely measurable sets. Again, we don't have to bother with SS in a positive measure space. Note that a measure space is (σ\sigma)-finite if and only if every measurable set is (σ\sigma)-finitely measurable. The finitely measurable sets form a δ\delta-ring, and the σ\sigma-finitely measurable sets form a σ\sigma-ring.

Recall that a Σ\Sigma-measurable function from (X,Σ)(X,\Sigma) to some other measurable space is any function ff such that the preimage under ff of a measurable set is always measurable (or something more complicated in the constructive theory). Now that we have a measure space, let a μ\mu-measurable function be a partial function ff from XX to some other measurable space such that the domain of ff is full and the preimage under ff of a measurable set is always μ\mu-measurable (that is measurable in the completion of μ\mu), and let two such functions be μ\mu-equivalent if their equaliser is a full set. We are really interested in the quotient set under this equivalence and so identify equivalent μ\mu-measurable functions. Classically, every μ\mu-measurable function is equivalent to some (total) Σ\Sigma-measurable function, so the definition is simpler in that case; however, partial functions still come up naturally in the classical theory, so it can be convenient to allow them rather than (as is usually done in a rigorous treatment) systematically replacing them with total functions.

A μ\mu-integrable function is a μ\mu-measurable function ff such that the integral ∫ Sfμ\int_S f \,\mu (as defined below) exists for every measurable set SS; it is enough to check S=XS = X. Equivalently, we may say that it is a μ\mu-measurable function ff such that the extended measure fμf \mu (also defined below) is actually a finite measure. (In any case, we get a finite measure fμf \mu if ff is integrable.)

Integration

In the following, ‘measurable’ will mean μ\mu-measurable. That is, we assume that μ\mu is complete and identify μ\mu-equivalent functions. We will also assume that μ\mu is a positive measure until I make sure of what must be done to generalise.

Given a measure μ\mu, a measurable set SS, and a measurable function ff, we will define the integral

∫ Sfμ \int_S f \,\mu

(see above for variations in notation) in stages, from the simplest form of ff to the most arbitrary.

Each measurable subset S⊆XS \subseteq X induces a measurable characteristic function χ S:X→ℝ +\chi_S\colon X \to \mathbb{R}_+ where χ S(x)=1\chi_S(x) = 1 if x∈Sx \in S, χ S(x)=0\chi_S(x) = 0 if x∈¬Sx \in \neg{S}. (In the constructive theory, where SS is a complemented pair, χ S\chi_S is a partial function with a full domain, so χ Sf\chi_S f is still a measurable function as defined above.) In general, we have

∫ Sfμ=∫ Xχ Sfμ, \int_S f \,\mu = \int_X \chi_S f \,\mu ,

so from now on we will assume that we are integrating over all of XX (and drop the subscript).

A positive simple function is a finite ℝ +\mathbb{R}_+-linear combination of measurable characteristic functions; the first form of integral that we define is

∫∑ 1≤i≤na iχ S iμ=∑ 1≤i≤na iμ(S i). \int \sum_{1 \leq i \leq n} a_i \chi_{S_i} \,\mu = \sum_{1 \leq i \leq n} a_i \mu(S_i) .

The integral is extended to all measurable functions f:X→[0,∞]f\colon X \to [0, \infty] by the rule

∫fμ=sup{∫sμ|0≤s≤f,ssimple} \int f \,\mu = \sup \{ \int s \,\mu \;|\; 0 \leq s \leq f, s\; simple \}

if this supremum converges. Classically, the integral either converges or diverges to infinity, so ∫fμ\int f \,\mu exists in some sense in any case; the possibilities are more complicated constructively.

For any measurable function f:X→[−∞,∞]f\colon X \to [-\infty, \infty], define f +f_+ and f −f_{-} by

f +(x)=max{f(x),0},f −(x)=max{−f(x),0} f_+(x) = \max\{f(x), 0\}, \qquad f_{-}(x) = \max\{-f(x), 0\}

so that f=f +−f −f = f_+ - f_{-}, |f|=f ++f −{|f|} = f_+ + f_{-}. Then the final definition is

∫fμ=∫f +μ−∫f −μ \int f \,\mu = \int f_{+} \,\mu - \int f_{-} \,\mu

if both integrals on the right converge. Classically, the other possibilities are ∞\infty, −∞-\infty, and ∞−∞\infty - \infty; not much can be done with the latter.

A measurable function ff is integrable with respect to μ\mu if this integral converges. It can be proved that all of the definitions above are consistent; that is, if the final definition is applied to a simple function, then it agrees with the original definition.

If ff takes values in the field ℂ\mathbb{C} of complex numbers or in some more general Banach space VV, then we can still ask whether |f|{|f|} is integrable. If it is, then we say that ff is absolutely integrable. We can then define the integral of ff; we always have

‖∫fμ‖≤∫‖f‖μ. {\|\int f \,\mu\|} \leq \int {\|f\|} \,\mu .

This integral is easy to define if VV has a basis; for example, a measurable complex-valued function f:X→ℂf\colon X \to \mathbb{C} is integrable iff both its real and imaginary parts are integrable, and we have

∫fμ=∫ℜfμ+i∫ℑfμ. \int f \,\mu = \int \Re{f} \,\mu + \mathrm{i} \int \Im{f} \,\mu .

The vector space of VV-valued integrable functions is itself a Banach space, using the norm

‖f‖ 1≔∫‖f‖μ. {\|f\|_1} \coloneqq \int {\|f\|} \,\mu .

Note that we must use the notion of measurable function as an equivalence class of functions to get a Banach space here; otherwise we have only a pre-Banach space (that is, a complete pseudonormed vector space).

This Banach space is called a Lebesgue space and is denoted L 1(μ,V)L^1(\mu,V), L 1(X,V)L^1(X,V), or just L 1L^1, depending on context. The default value of VV is usually either ℝ\mathbb{R} or ℂ\mathbb{C}, depending on the author. More general Lebesgue spaces of the form L pL^p also exist; ff is in L pL^p precisely when |f| p{|f|^p} is integrable, and we use

‖f‖ p≔∫‖f‖ pμp {\|f\|_p} \coloneqq \root p {\int {\|f\|^p} \,\mu}

as the norm. (At least, this is so for p∈]0,∞[p \in {]{0,\infty}[}; see Lebesgue space for generalizations to other values of pp.)

The algebra of measures

Note that

(fμ)(S)=∫χ Sfμ (f \mu) (S) = \int \chi_S f \,\mu

makes fμf \mu into a VV-valued measure whenever ff is an integrable VV-valued function. When ff is [−∞,∞][-\infty,\infty]-valued and μ\mu is a signed measure, then ff is an extended measure which is finite iff ff is integrable. We have

(fg)μ=f(gμ). (f g) \mu = f (g \mu) .

Thus integration can be seen as a way of multiplying a function by a measure to get another measure.

The Radon-Nikodym derivative is about reversing this (dividing two measures to get a function).

Noncommutative measure theory

Every commutative von Neumann algebra is isomorphic to the Lebesgue space L ∞(X,μ)L^\infty(X,\mu) where μ\mu is some measure (which is irrelevant) on a localisable measurable space, and this extends to a duality between localisable measurable spaces and commutative von Neumann algebras. This is similar to the correspondence between commutative C *C^*-algebras and locally compact Hausdorff spaces, which is the central approach to noncommutative geometry. It is useful to exploit the intuition that the theory of (noncommutative) von Neumann algebras is a noncommutative analogue of classical measure theory.
