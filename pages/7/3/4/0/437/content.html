Idea

A path space object in homotopy theory is an object in a homotopical category that behaves for many purposes as the topological path space does in topological homotopy theory.

(in model categories) In any model category, the factorization axioms applied to the diagonal maps immediately imply that every object has a path space object, and in fact a ‚Äúvery good‚Äù one. (See below.)

The very good path space objects in locally cartesian closed model categories serve as categorical semantics for the identity types in dependent type theory (homotopy type theory).

Related notions

Right homotopies

Path objects are used to define a notion of right homotopy between morphisms in a category. Thus they capture aspects of higher category theory in a 11-categorical context.

Loop space objects

From a path space object may be derived loop space objects.
