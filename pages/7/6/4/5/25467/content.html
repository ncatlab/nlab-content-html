Idea

Suppose we have a fibered category p:ℰ→ℬp \colon \mathcal{E} \to \mathcal{B}, representing type families (living in the fibers of ℰ\mathcal{E}) indexed by contexts (living in ℬ\mathcal{B}).

A definable collection of objects of the total category ℰ\mathcal{E} corresponds, intuitively, to the extension of a ‘meta-property’ of the types which we can use to single out subcontexts by means of a generalized ‘comprehension’.

The archetypal example is given by ‘truth’. Suppose pp has fibred terminal objects?, i.e. over each context B:ℬB:\mathcal{B}, there is a ‘unit type’ 1 B:ℰ B1_B : \mathcal{E}_B, terminal in the fiber. Then the collection {1 B:ℰ∣B:ℬ}\{1_B : \mathcal{E} \mid B: \mathcal{B}\} is an extensive definition of the meta-property of ‘being true’ (or better: being inhabited).

Once we have a definable collection 𝒫\mathcal{P}, and given an object X:ℰ BX: \mathcal{E}_B, we can ask: which members of the ‘family’ XX ‘satisfy’ PP? This question is answered by a subobject {b:B∣X b:𝒫}↣B:ℬ\{b:B \mid X_b : \mathcal{P}\} \rightarrowtail B : \mathcal{B}.

In a sense, a definable collection is a very weak notion of coreflective subfibration (i.e. a fibred coreflective subcategory).

Definition

We follow the exposition in (Jacobs ‘99). Fix a fibration p:ℰ→ℬp:\mathcal{E} \to \mathcal{B}.

Definition

Let 𝒫\mathcal{P} be a collection of object in ℰ\mathcal{E}, and denote by 𝒫 B\mathcal{P}_B the subcollection of those objects in 𝒫\mathcal{P} over a chosen B:ℬB:\mathcal{B}.

  1. We say 𝒫\mathcal{P} is closed under substitution if for each map u:J→Iu:J \to I in ℬ\mathcal{B} and P:𝒫 IP : \mathcal{P}_I, u *P:𝒫 Ju^*P : \mathcal{P}_J.

  2. We say 𝒫\mathcal{P} is definable if it is closed under substitution and, for each object X:ℰX: \mathcal{E}, there is an object {X} 𝒫:𝒫\{X\}_{\mathcal{P}} : \mathcal{P} and a cartesian morphism i X:{X} 𝒫→Xi_X : \{X\}_{\mathcal{P}} \to X which is universal among cartesian morphisms into XX from an object in 𝒫\mathcal{P}.

The constructor {−} 𝒫\{-\}_{\mathcal{P}} is the generalized comprehension we were mentioning above. Indeed, we have the following characterization:

Lemma

(Lemma 9.6.2 in (Jacobs ‘99))

Let 𝒫\mathcal{P} be a collection closed under substitution. Then the following our equivalent:

  1. 𝒫\mathcal{P} is definable
  2. For each object X:ℰ BX: \mathcal{E}_B the presheaf on ℬ\mathcal{B} defined as
    J↦{u:J→I∣u *X:𝒫} J \mapsto \{u : J \to I \mid u^*X : \mathcal{P}\}

    is representable.

Proof

The idea is that {X} P\{X\}_P is the representing object for the presheaf object of (2).

Examples

Example

If pp has a fibred terminal objects, then their collection is definable if and only if the functor 1:ℬ→ℰ1: \mathcal{B} \to \mathcal{E} has a right adjoint {−}:ℰ→ℬ\{-\}:\mathcal{E} \to \mathcal{B}.

Example

For p:Fam(𝒞)→Setp : \mathrm{Fam}(\mathcal{C}) \to \mathbf{Set} a fibration of families (a family over I:SetI:\mathbf{Set} is a n II-indexed family of objects of 𝒞\mathcal{C}), definable collection are those determined by a choice of objects of 𝒞\mathcal{C} (Lemma 9.6.4 in in (Jacobs ‘99)).
