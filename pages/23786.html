Idea

An extension of a fragment of first order Peano arithmetic as described in Edward Nelson’s paper.

 Definition

Nelson arithmetic is done by extending the fragment of first order Peano arithmetic consisting of the following axioms:

  1. ∀ x¬s(x)=0\forall_x \neg s(x) = 0;

  2. ∀ x,y(s(x)=s(y))⇒(x=y)\forall_{x, y} (s(x) = s(y)) \Rightarrow (x = y);

  3. ∀ xx+0=x\forall_x x + 0 = x,

  4. ∀ x,yx+s(y)=s(x+y)\forall_{x, y} x + s(y) = s(x+y);

  5. ∀ xx⋅0=0\forall_x x \cdot 0 = 0;

  6. ∀ x,yx⋅s(y)=x⋅y+x\forall_{x, y} x \cdot s(y) = x \cdot y + x;

with a predicate 𝒞\mathcal{C} satisfying the following axioms

  1. 𝒞(0)\mathcal{C}(0);

  2. ∀ x𝒞(x)⇒𝒞(s(x))\forall_{x} \mathcal{C}(x) \Rightarrow \mathcal{C}(s(x));

Models

We work in a dependent type theory with identity types, empty type, unit type, and booleans. A model of Nelson arithmetic is a type NN with

  • an element 0:N0:N,
  • a function s:N→Ns:N \to N,
  • a function (−)+(−):N×N→N(-)+(-):N \times N \to N,
  • a function (−)⋅(−):N×N→N(-)\cdot(-):N \times N \to N,
  • a type family 𝒞(−)\mathcal{C}(-)
  • a dependent function
    α:∏ x:N∏ a:𝒞(x)∏ b:𝒞(x)a= 𝒞(x)b\alpha: \prod_{x:N} \prod_{a:\mathcal{C}(x)} \prod_{b:\mathcal{C}(x)} a =_{\mathcal{C}(x)} b
  • a dependent function
    β:∏ x:N(s(x)=0)→𝟘\beta: \prod_{x:N} (s(x) = 0) \to \mathbb{0}
  • a dependent function
    γ:∏ x:N∏ y:Ns(x)=s(y)→x=y\gamma: \prod_{x:N} \prod_{y:N} s(x) = s(y) \to x = y
  • a dependent function
    δ:∏ x:Nx+0=x\delta: \prod_{x:N} x + 0 = x
  • a dependent function
    ϵ:∏ x:N∏ y:Nx+s(y)=s(x+y)\epsilon: \prod_{x:N} \prod_{y:N} x + s(y) = s(x+y)
  • a dependent function
    ζ:∏ x:Nx⋅0=0\zeta: \prod_{x:N} x \cdot 0 = 0
  • a dependent function
    η:∏ x:N∏ y:Nx⋅s(y)=x⋅y+x\eta: \prod_{x:N} \prod_{y:N} x \cdot s(y) = x \cdot y + x
  • a term
    ι:𝒞(0)\iota: \mathcal{C}(0)
  • a dependent function
    κ:∏ x:N𝒞(x)→𝒞(s(x))\kappa: \prod_{x:N} \mathcal{C}(x) \to \mathcal{C}(s(x))
  • a dependent function
    τ 0:∏ x:N∏ y:N∏ a:x= Ny∏ b:x= Nya= x=yb\tau_0: \prod_{x:N} \prod_{y:N} \prod_{a:x =_N y} \prod_{b:x =_N y} a =_{x = y} b

A homomorphism of models of Nelson arithmetic NN and N ′N^{'} is a function f:N→N ′f:N \to N^{'} with

  • a dependent function
    f 𝒞:∏ x:N𝒞(x)→𝒞(f(x))f_\mathcal{C}:\prod_{x:N} \mathcal{C}(x) \to \mathcal{C}(f(x))
  • a term
    a:f(0)=0a:f(0) = 0
  • a dependent function
    b:∏ x:Nf(s(x))=s(f(x))b:\prod_{x:N} f(s(x)) = s(f(x))
  • a dependent function
    c:∏ x:N∏ y:Nf(x+y)=f(x)+f(y)c:\prod_{x:N} \prod_{y:N} f(x + y) = f(x) + f(y)
  • a dependent function
    d:∏ x:N∏ y:Nf(x⋅y)=f(x)⋅f(y)d:\prod_{x:N} \prod_{y:N} f(x \cdot y) = f(x) \cdot f(y)
  • a dependent function
