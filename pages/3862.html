Quillen‚Äôs determinant line bundle

There is a specific version called Quillen‚Äôs determinant line bundle which is certain line bundle over the moduli space of complex structures on a fixed smooth vector bundle EE over a fixed Riemann surface MM. A complex structure on the bundle corresponds to an operator which in local coordinates looks as D=dz¬Ø(‚àÇ z+Œ±(z))D = d\bar{z}(\partial_z+\alpha(z)) where Œ±(z)\alpha(z) is a smooth matrix valued function. Then again a determinant is an element of a line ‚Ñí D=Œª(KerD) *‚äóŒª(CokerD)\mathcal{L}_D = \lambda(Ker D)^*\otimes \lambda(Coker D) where Œª\lambda is taking the top exterior power. If we had a trivialization of the Quillen‚Äôs determinant line bundle, then we could identify every section with a holomorphic function on the base space, hence a holomorphic rule giving a number to a Cauchy-Riemann operator. For this one restricts first to the component consisting of the operators with the zero Fredholm index. Next, one considers the corresponding Laplace operator D *DD^* D and its functional determinant related to the zeta function of an elliptic differential operator. (This is related to the analytic torsion).
